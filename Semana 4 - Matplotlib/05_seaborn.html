¿Qué es Seaborn y sus Ventajas?

Seaborn es una librería de visualización de datos de Python basada en Matplotlib. Proporciona una interfaz de alto nivel para dibujar gráficos estadísticos atractivos e informativos.

Ventajas sobre Matplotlib (para ciertos usos):

  • Integración con Pandas: Está diseñada para trabajar muy bien con DataFrames de Pandas. Muchas funciones de Seaborn aceptan DataFrames directamente y puedes especificar columnas por sus nombres.
  • Estilos Atractivos por Defecto: Los gráficos de Seaborn suelen tener una apariencia más moderna y estéticamente agradable "de fábrica" que los gráficos base de Matplotlib.
  • Funciones para Gráficos Estadísticos Complejos: Facilita la creación de visualizaciones estadísticas comunes (como distribuciones, relaciones, datos categóricos) que requerirían más código en Matplotlib puro.
  • Menos Código para Gráficos Comunes: A menudo puedes lograr gráficos sofisticados con menos líneas de código.

Piensa en Seaborn como una capa sobre Matplotlib que simplifica la creación de ciertos tipos de gráficos estadísticos y mejora su apariencia.

Por convención, Seaborn se importa con el alias sns. También es común importar Matplotlib para ajustes finos. 
import seaborn as sns
import matplotlib.pyplot as plt # A menudo se usa junto con matplotlib
import pandas as pd    # Para los DataFrames

Gráficos de Distribución

Seaborn sobresale en la visualización de distribuciones de datos.

sns.histplot(data=df, x='columna', kde=True): Similar al histograma de Matplotlib, pero puede añadir una estimación de la densidad del kernel (KDE) para suavizar la distribución.

sns.boxplot(data=df, x='categoria', y='valor'): Muestra la distribución de datos numéricos a través de diferentes categorías usando diagramas de caja y bigotes (cuartiles, mediana, valores atípicos).

sns.violinplot(data=df, x='categoria', y='valor'): Similar al boxplot, pero también muestra la forma de la distribución (KDE) a cada lado de la caja.

import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd
import numpy as np

# Datos de ejemplo
np.random.seed(10)
datos_dist = pd.DataFrame({
    'Valor': np.random.randn(100),
    'Categoria': np.random.choice(['A', 'B', 'C'], 100)
})

plt.figure(figsize=(12, 4))

plt.subplot(1, 3, 1)
sns.histplot(data=datos_dist, x='Valor', kde=True)
plt.title('Histograma con KDE')

plt.subplot(1, 3, 2)
sns.boxplot(data=datos_dist, x='Categoria', y='Valor')
plt.title('Boxplot por Categoría')

plt.subplot(1, 3, 3)
sns.violinplot(data=datos_dist, x='Categoria', y='Valor')
plt.title('Violinplot por Categoría')

plt.tight_layout()
# plt.show()
print("Gráficos de Distribución generados (simulados).")
(Simulación de 3 subplots en una fila)
1. Histograma de 'Valor' con curva KDE.
2. Boxplots de 'Valor' para cada 'Categoria' (A, B, C).
3. Violinplots de 'Valor' para cada 'Categoria' (A, B, C).

Gráficos Categóricos: countplot()

Para visualizar la frecuencia o el conteo de observaciones en diferentes categorías, sns.countplot() es muy útil. Es como un histograma para datos categóricos.

Sintaxis: sns.countplot(data=df, x='columna_categorica') o y='columna_categorica' para barras horizontales.

import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd

data_cat = {'Dia': ['Lunes', 'Martes', 'Lunes', 'Miércoles', 'Lunes', 'Martes']}
df_cat = pd.DataFrame(data_cat)

plt.figure()
sns.countplot(data=df_cat, x='Dia', palette='viridis')
plt.title('Conteo de Ocurrencias por Día')
plt.ylabel('Frecuencia')
# plt.show()
print("Countplot generado (simulado).")
Conteo de Ocurrencias por Día (Título)
Eje X: Día (Lunes, Martes, Miércoles) | Eje Y: Frecuencia
(Barras mostrando conteos: Lunes=3, Martes=2, Miércoles=1. Colores de la paleta 'viridis')

Parámetros Comunes: hue y palette

Muchos gráficos de Seaborn aceptan parámetros para añadir dimensiones adicionales de información a través del color y otros atributos visuales.

  • hue: Toma el nombre de una columna. Los diferentes valores de esta columna se usarán para colorear o diferenciar elementos del gráfico (ej. diferentes barras, puntos, líneas). Esto permite comparar distribuciones o relaciones para diferentes subgrupos.
  • palette: Especifica la paleta de colores a usar cuando se usa hue. Seaborn tiene muchas paletas predefinidas (ej. 'viridis', 'coolwarm', 'Set2', 'pastel').

Ejemplo con boxplot y hue:

import seaborn as sns
import matplotlib.pyplot as plt
import pandas as pd
import numpy as np

# Usaremos los datos_dist del paso anterior
np.random.seed(10)
datos_dist = pd.DataFrame({
    'Valor': np.random.randn(100),
    'Categoria': np.random.choice(['A', 'B', 'C'], 100),
    'Subgrupo': np.random.choice(['S1', 'S2'], 100)
})

plt.figure(figsize=(8, 5))
sns.boxplot(data=datos_dist, x='Categoria', y='Valor', hue='Subgrupo', palette='pastel')
plt.title('Boxplot por Categoría y Subgrupo')
# plt.show()
print("Boxplot con 'hue' generado (simulado).")
Boxplot por Categoría y Subgrupo (Título)
Eje X: Categoría (A, B, C) | Eje Y: Valor
(Para cada categoría, dos boxplots lado a lado, uno para Subgrupo S1 y otro para S2, con colores pastel)

Otros parámetros comunes son data (el DataFrame), x (columna para el eje x), y (columna para el eje y).

Resumen: Introducción a Seaborn

Seaborn simplifica la creación de gráficos estadísticos atractivos.

🌟 ¿Qué es Seaborn?
Librería de visualización estadística sobre Matplotlib. import seaborn as sns
🐼 Integración Pandas
Funciona directamente con DataFrames.
🧬 histplot
Histogramas (con KDE opcional). sns.histplot(data, x, kde)
🎻 boxplot / violinplot
Distribuciones por categoría. sns.boxplot(data, x, y)
☰ countplot
Frecuencia de categorías. sns.countplot(data, x)
🎨 hue / palette
Añade otra dimensión con color.
⚙️ Parámetros
data, x, y son comunes para especificar datos y ejes.

Recuerda que Seaborn usa Matplotlib por debajo, así que puedes usar funciones de plt (como plt.title()) para personalizar aún más.
