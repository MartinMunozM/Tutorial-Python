¿Qué es Matplotlib?

Matplotlib es una librería de Python muy popular y completa para crear visualizaciones estáticas, animadas e interactivas.

Es la base de muchas otras librerías de visualización en Python (como Seaborn, que veremos más adelante) y ofrece un control detallado sobre cada aspecto de un gráfico.

¿Por qué usar Matplotlib?

  • Versatilidad: Permite crear una amplia gama de gráficos: líneas, barras, histogramas, dispersión, etc.
  • Personalización: Ofrece un control granular sobre casi todos los elementos de un gráfico (colores, etiquetas, fuentes, leyendas, etc.).
  • Integración: Funciona muy bien con NumPy, Pandas y el ecosistema científico de Python.
  • Calidad de Publicación: Puede generar gráficos de alta calidad adecuados para publicaciones e informes.

El módulo más comúnmente usado de Matplotlib es pyplot, que proporciona una interfaz similar a MATLAB para crear gráficos.

Por convención, matplotlib.pyplot se importa con el alias plt: 
import matplotlib.pyplot as plt
🖼️

Dos Enfoques: pyplot vs. Figura/Ejes

Hay dos formas principales de usar Matplotlib:

  1. Interfaz pyplot (estilo MATLAB):
    • Es una colección de funciones que operan sobre una figura y ejes "globales" o actuales.
    • Es más rápido para gráficos simples y exploración rápida.
    • Ejemplo: plt.plot(...), plt.title(...).
  2. Interfaz Orientada a Objetos (OO):
    • Se crean explícitamente objetos Figure (la ventana o lienzo completo) y Axes (los subgráficos individuales donde se dibujan los datos).
    • Ofrece más control y flexibilidad, especialmente para gráficos complejos o múltiples subplots.
    • Ejemplo: fig, ax = plt.subplots(), luego ax.plot(...), ax.set_title(...).

    3. Para empezar, la interfaz pyplot es más sencilla. A medida que necesites más control, la interfaz OO se vuelve más útil.

    import matplotlib.pyplot as plt
import numpy as np

# Datos de ejemplo
x = np.array([1, 2, 3, 4])
y = np.array([2, 4, 1, 5])

# Enfoque pyplot
plt.figure() # Opcional, crea una nueva figura
plt.plot(x, y)
plt.title("Gráfico con pyplot")
# plt.show() # Mostraría el gráfico

# Enfoque Orientado a Objetos
fig, ax = plt.subplots() # Crea una figura y un conjunto de ejes
ax.plot(x, y)
ax.set_title("Gráfico con Figura y Ejes")
# plt.show() # Mostraría este otro gráfico

    Ambos enfoques producen resultados similares para gráficos simples. La interfaz OO es generalmente recomendada para scripts y aplicaciones más grandes.

Gráficos de Líneas: plt.plot()

La función más básica para crear gráficos es plt.plot(). Por defecto, si le pasas una sola secuencia de números, los tomará como los valores del eje Y y generará automáticamente los valores del eje X (0, 1, 2...).

Si pasas dos secuencias, la primera se usa para el eje X y la segunda para el eje Y.

import matplotlib.pyplot as plt
import numpy as np

y_valores = np.array([3, 8, 1, 10, 5])

plt.figure() # Crea una nueva figura para el gráfico
plt.plot(y_valores)
plt.title("Gráfico de Línea Simple (solo Y)")
# plt.show() # En un script real, esto mostraría el gráfico
print("Gráfico 1 (solo Y) generado (simulado).")
Gráfico de Línea Simple (solo Y) (Línea conectando puntos en Y: 3, 8, 1, 10, 5 con X: 0, 1, 2, 3, 4)

Con valores X e Y:

x_valores = np.array([1, 2, 3, 4, 5])
y_valores_2 = np.array([2, 3, 5, 7, 11]) # Números primos

plt.figure() # Otra figura
plt.plot(x_valores, y_valores_2)
plt.title("Gráfico de Línea con X e Y")
# plt.show()
print("\nGráfico 2 (X e Y) generado (simulado).")
Gráfico de Línea con X e Y (Línea conectando (1,2), (2,3), (3,5), (4,7), (5,11))

Mostrar Gráficos y Personalización

Mostrando Gráficos con plt.show():

Después de definir tu gráfico (o gráficos) usando funciones de pyplot, necesitas llamar a plt.show() para que realmente se muestren en una ventana.

En entornos como Jupyter Notebooks, a veces los gráficos se muestran automáticamente después de la celda que los genera (gracias a "magias" como %matplotlib inline). Sin embargo, en scripts de Python (.py), plt.show() es esencial.

En este tutorial interactivo web, no podemos ejecutar plt.show() para mostrar ventanas emergentes. Usaremos marcadores de posición para simular la apariencia de los gráficos.

Personalización Básica: Color y Estilo de Línea

La función plt.plot() acepta argumentos adicionales para personalizar la apariencia. Los más comunes son:

  • color: Especifica el color de la línea (ej. 'red', 'blue', 'g' para verde, '#FF5733' para un color hexadecimal).
  • linestyle (o ls): Define el estilo de la línea (ej. '-' para sólida (default), '--' para discontinua, ':' para punteada, '-.' para guion-punto).
  • linewidth (o lw): Grosor de la línea.
  • marker: Estilo del marcador para cada punto de datos (ej. 'o' para círculo, '.' para punto, '*' para estrella, 's' para cuadrado).

También puedes usar un formato de cadena abreviado que combina color, marcador y estilo de línea (ej. 'ro-' para rojo, círculo, línea sólida).

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 100) # 100 puntos entre 0 y 10
y1 = np.sin(x)
y2 = np.cos(x)

plt.figure(figsize=(8, 4)) # Opcional: tamaño de la figura

# Graficar y1 con línea roja discontinua y marcadores 'o'
plt.plot(x, y1, color='red', linestyle='--', marker='o', label='Seno')

# Graficar y2 con formato abreviado: línea azul punteada con estrellas
plt.plot(x, y2, 'b:*', label='Coseno') # b: blue, :: dotted, *: star marker

plt.title("Funciones Seno y Coseno")
plt.xlabel("Eje X")
plt.ylabel("Eje Y")
plt.legend() # Muestra la leyenda (usa los 'label' de plot)
# plt.grid(True) # Opcional: añade una cuadrícula
# plt.show()
print("Gráfico personalizado generado (simulado).")
Funciones Seno y Coseno (Seno: línea roja discontinua con círculos) (Coseno: línea azul punteada con estrellas) Etiquetas X, Y y Leyenda presentes.
