Títulos y Etiquetas (title, xlabel, ylabel)

Un gráfico sin contexto es difícil de entender. Es crucial añadir un título al gráfico y etiquetas a los ejes X e Y para describir qué representan los datos.

En la interfaz pyplot, usamos:

  • plt.title("Mi Título Increíble"): Añade un título principal al gráfico.
  • plt.xlabel("Etiqueta del Eje X"): Etiqueta el eje horizontal.
  • plt.ylabel("Etiqueta del Eje Y"): Etiqueta el eje vertical.

Puedes personalizar la fuente, tamaño y color de estos textos con argumentos adicionales (ej. fontsize=14, color='blue').

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 2 * np.pi, 100)
y = np.sin(x)

plt.figure()
plt.plot(x, y)

plt.title("Función Seno", fontsize=16, color='darkblue')
plt.xlabel("Ángulo (radianes)", fontsize=12)
plt.ylabel("Valor del Seno", fontsize=12)

# plt.show() # Para mostrar el gráfico
print("Gráfico con Título y Etiquetas generado (simulado).")
Función Seno (Título, Azul Oscuro, Tamaño 16)
Eje X: Ángulo (radianes) (Tamaño 12)
Eje Y: Valor del Seno (Tamaño 12)
(Gráfico de la función seno de 0 a 2π)

Añadiendo Leyendas con legend()

Cuando graficas múltiples líneas o conjuntos de datos en un mismo gráfico, una leyenda es esencial para identificar qué representa cada uno.

Para crear una leyenda:

  1. Añade el argumento label="Nombre de la Serie" a cada llamada de plt.plot() (o función de graficación similar).
  2. Llama a plt.legend() después de todas las funciones de graficación.

plt.legend() tiene parámetros para controlar la posición (loc), el tamaño de la fuente, etc.

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(0, 10, 50)
y_cuadrado = x ** 2
y_cubo = x ** 3

plt.figure()
plt.plot(x, y_cuadrado, label='x^2 (Cuadrado)', color='blue')
plt.plot(x, y_cubo, label='x^3 (Cubo)', color='red', linestyle='--')

plt.title("Funciones Potenciales")
plt.xlabel("Base x")
plt.ylabel("Resultado Potencia")
plt.legend(loc='upper left') # Ubicación de la leyenda

# plt.show()
print("Gráfico con Leyenda generado (simulado).")
Funciones Potenciales (Título)
Eje X: Base x | Eje Y: Resultado Potencia
(Línea azul para x^2, Línea roja discontinua para x^3)
Leyenda en la esquina superior izquierda mostrando 'x^2 (Cuadrado)' y 'x^3 (Cubo)'.

Colores, Estilos y Marcadores

Ya vimos brevemente cómo cambiar el color, estilo de línea y marcadores. Matplotlib ofrece una gran variedad de opciones.

Colores Comunes:

  • Nombres: 'blue', 'green', 'red', 'cyan', 'magenta', 'yellow', 'black', 'white'.
  • Abreviaturas: 'b', 'g', 'r', 'c', 'm', 'y', 'k', 'w'.
  • Valores hexadecimales RGB: '#RRGGBB' (ej. '#FF5733').
  • Tuplas RGB/RGBA: (R, G, B) o (R, G, B, A) con valores entre 0 y 1.

Estilos de Línea Comunes (linestyle o ls):

  • '-' o 'solid' (sólida)
  • '--' o 'dashed' (discontinua)
  • ':' o 'dotted' (punteada)
  • '-.' o 'dashdot' (guion-punto)
  • 'None' o '' (sin línea, solo marcadores si se especifican)

Marcadores Comunes (marker):

  • '.' (punto), ',' (pixel)
  • 'o' (círculo), 'v' (triángulo abajo), '^' (triángulo arriba)
  • 's' (cuadrado), 'p' (pentágono), '*' (estrella), '+' (más), 'x' (equis)
  • 'D' (diamante), 'd' (diamante delgado)

También puedes controlar el tamaño del marcador (markersize), el color del borde del marcador (markeredgecolor), y el color de relleno del marcador (markerfacecolor).

import matplotlib.pyplot as plt
import numpy as np

x = np.arange(5)
plt.figure()

plt.plot(x, x+1, color='green', linestyle=':', marker='s', label='Verde Punteado Cuadrado')
plt.plot(x, x+2, 'm--*', linewidth=2, markersize=10, label='Magenta Discontinua Estrella')
plt.plot(x, x+3, color='#FFC300', linestyle='-.', marker='.', label='Amarillo GuionPunto Punto')

plt.legend()
# plt.show()
print("Gráfico con múltiples estilos generado (simulado).")
Gráfico con tres líneas: 1. Verde punteada con marcadores cuadrados. 2. Magenta discontinua gruesa con marcadores de estrella grandes. 3. Amarilla (hex #FFC300) guion-punto con marcadores de punto. Leyenda visible.

Guardando Figuras: savefig()

Una vez que has creado un gráfico, querrás guardarlo en un archivo para usarlo en informes, presentaciones, o en la web.

Matplotlib usa la función plt.savefig() para esto.

Sintaxis: plt.savefig('nombre_del_archivo.extension', dpi=300, bbox_inches='tight')

  • El primer argumento es el nombre del archivo y su extensión (ej. .png, .jpg, .pdf, .svg). Matplotlib infiere el formato a partir de la extensión.
  • dpi (dots per inch): Controla la resolución de la imagen (para formatos raster como PNG, JPG). Valores más altos significan mayor resolución.
  • bbox_inches='tight': A menudo útil para asegurar que todo el contenido del gráfico (incluyendo títulos y etiquetas) se guarde sin ser recortado.
  • Otros parámetros incluyen facecolor (color de fondo de la figura), transparent=True, etc.
import matplotlib.pyplot as plt
import numpy as np

x = np.random.rand(50)
y = np.random.rand(50)
colores = np.random.rand(50)
tamanos = 1000 * np.random.rand(50)

plt.figure()
plt.scatter(x, y, c=colores, s=tamanos, alpha=0.5, cmap='viridis')
plt.title("Gráfico de Dispersión Aleatorio")
plt.colorbar(label='Intensidad de Color') # Añade una barra de color

# Guardar la figura (esto crearía el archivo si se ejecutara)
# plt.savefig('mi_grafico_dispersion.png', dpi=300, bbox_inches='tight')
print("Simulación: Gráfico guardado como 'mi_grafico_dispersion.png'.")
# plt.show() # Mostraría el gráfico también
Gráfico de Dispersión Aleatorio
(Puntos dispersos con diferentes colores y tamaños)
Barra de color presente.
Es importante llamar a plt.savefig() antes de plt.show() si estás en un script. Después de plt.show(), la figura a menudo se "limpia" y savefig() podría guardar una imagen en blanco.
