Tutorial: Subplots y Organización de Figuras

Crea visualizaciones complejas con múltiples gráficos en Matplotlib

¿Por qué Múltiples Gráficos (Subplots)?

A menudo, necesitas mostrar múltiples gráficos relacionados en una sola figura para comparar diferentes aspectos de tus datos, mostrar diferentes variables, o presentar una historia visual más completa.

Matplotlib te permite crear una figura que contenga una cuadrícula de "subgráficos" (llamados Axes en la terminología de Matplotlib). Cada subgráfico puede tener su propio tipo de gráfico, datos y personalizaciones.

Esto es fundamental para:

  • Comparar tendencias entre diferentes conjuntos de datos.
  • Mostrar diferentes vistas de los mismos datos (ej. datos crudos vs. datos suavizados).
  • Crear dashboards o informes visuales compactos.

La forma más común y flexible de trabajar con subplots es usando la interfaz orientada a objetos de Matplotlib.

🖼️

Creando Subplots: plt.subplots()

La función plt.subplots(nrows, ncols) es la forma recomendada para crear una figura y un conjunto de subplots (ejes) en una cuadrícula.

  • nrows: Número de filas de subplots.
  • ncols: Número de columnas de subplots.

Devuelve dos objetos:

  1. Un objeto Figure (fig): El lienzo completo donde se dibujarán todos los subplots.
  2. Un array de objetos Axes (ax o axes): Cada elemento de este array representa un subgráfico individual.
    • Si creas un solo subplot (ej. plt.subplots(1,1), que es lo mismo que plt.subplots()), ax será un solo objeto Axes.
    • Si creas múltiples subplots (ej. plt.subplots(2,2)), axes será un array NumPy 2D de objetos Axes.

Una vez que tienes un objeto Axes (ej. ax), usas sus métodos para dibujar en él (ej. ax.plot(), ax.set_title()) en lugar de las funciones globales de plt.

import matplotlib.pyplot as plt
import numpy as np

# Crear una figura con 2 filas y 1 columna de subplots
fig, axes = plt.subplots(nrows=2, ncols=1, figsize=(6, 6)) # axes será un array de 2 Axes

x = np.linspace(0, 10, 100)

# Graficar en el primer subplot (axes[0])
axes[0].plot(x, np.sin(x), color='blue')
axes[0].set_title('Función Seno')

# Graficar en el segundo subplot (axes[1])
axes[1].plot(x, np.cos(x), color='red')
axes[1].set_title('Función Coseno')
axes[1].set_xlabel('X')

fig.tight_layout() # Ajusta el espaciado para evitar superposiciones
# plt.show()
print("Figura con 2 subplots generada (simulada).")
Figura con 2 Subplots (2 filas, 1 columna)
--- Subplot 1 (Superior) ---
Título: Función Seno
(Gráfico de línea azul de la función seno)
--- Subplot 2 (Inferior) ---
Título: Función Coseno
Eje X: X
(Gráfico de línea roja de la función coseno)

Iteración para Múltiples Subgráficos

Cuando tienes una cuadrícula de subplots (ej. 2x2, 3x2), plt.subplots() devuelve un array 2D de objetos Axes. Puedes iterar sobre este array para graficar en cada subgráfico.

Si el array de Axes es 1D (ej. plt.subplots(3,1)), puedes iterar directamente. Si es 2D, puedes usar bucles anidados o el método .flatten() para convertirlo en 1D y luego iterar.

import matplotlib.pyplot as plt
import numpy as np

# Crear una figura 2x2
fig, axes = plt.subplots(2, 2, figsize=(8, 6)) # axes es un array 2x2 de Axes

x = np.linspace(0, 5, 50)
colores = ['blue', 'green', 'red', 'purple']
funciones = [np.sin, np.cos, np.tan, lambda val: val**2]
titulos = ['Seno', 'Coseno', 'Tangente', 'Cuadrado']

for i, ax in enumerate(axes.flatten()): # Aplanar el array 2D de Axes a 1D
    ax.plot(x, funciones[i](x), color=colores[i])
    ax.set_title(titulos[i])
    if i >= 2: # Añadir etiqueta X a la fila inferior
        ax.set_xlabel('x')
    if i % 2 == 0: # Añadir etiqueta Y a la columna izquierda
        ax.set_ylabel('y')

fig.suptitle('Múltiples Funciones en Subplots', fontsize=16) # Título para toda la figura
fig.tight_layout(rect=[0, 0, 1, 0.96]) # Ajustar para dejar espacio al supertítulo
# plt.show()
print("Figura 2x2 con iteración generada (simulada).")
Múltiples Funciones en Subplots (Supertítulo)
--- Cuadrícula 2x2 ---
(0,0) Seno (azul), (0,1) Coseno (verde)
(1,0) Tangente (rojo), (1,1) Cuadrado (púrpura)
Etiquetas X en fila inferior, Etiquetas Y en columna izquierda.

Ajuste de Espacio y Gridspec

Ajuste Automático con fig.tight_layout():

Cuando tienes múltiples subplots con títulos, etiquetas, etc., a veces se superponen. La función fig.tight_layout() ajusta automáticamente los parámetros de los subplots para dar un "ajuste apretado" y evitar superposiciones.

A menudo se llama justo antes de plt.show().

# ... código para crear fig y axes ...
fig.tight_layout() 
# plt.show()

Control Más Fino con GridSpec:

Para diseños de subplots más complejos (ej. un subplot que ocupe varias celdas de la cuadrícula), puedes usar matplotlib.gridspec.GridSpec.

import matplotlib.pyplot as plt
import matplotlib.gridspec as gridspec
import numpy as np

fig = plt.figure(figsize=(8, 6))
gs = gridspec.GridSpec(2, 2, width_ratios=[2, 1], height_ratios=[1, 2])

ax1 = fig.add_subplot(gs[0, :]) # Ocupa toda la primera fila
ax1.plot(np.random.rand(10))
ax1.set_title('Gráfico Superior Ancho')

ax2 = fig.add_subplot(gs[1, 0]) # Segunda fila, primera columna (más ancha)
ax2.hist(np.random.randn(100), bins=10)
ax2.set_title('Histograma')

ax3 = fig.add_subplot(gs[1, 1]) # Segunda fila, segunda columna
ax3.scatter(np.random.rand(10), np.random.rand(10))
ax3.set_title('Dispersión')

fig.tight_layout()
# plt.show()
print("Figura con Gridspec generada (simulada).")
Figura con Diseño GridSpec
--- Gráfico Superior Ancho (ocupa 2 columnas en fila 0) ---
(Gráfico de línea aleatorio)
--- Fila Inferior ---
(Col Izquierda, más ancha) Histograma | (Col Derecha) Dispersión
Gridspec es más avanzado pero ofrece control total sobre la disposición de subplots de diferentes tamaños y proporciones.
