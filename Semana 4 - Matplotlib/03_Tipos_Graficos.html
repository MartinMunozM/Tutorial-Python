Gráficos de Barras: plt.bar()

Los gráficos de barras son excelentes para comparar cantidades entre diferentes categorías.

Se crean con plt.bar(categorias, valores).

  • El primer argumento son las posiciones o etiquetas de las barras en el eje x.
  • El segundo argumento son las alturas de las barras.
  • Puedes usar plt.barh() para barras horizontales.
import matplotlib.pyplot as plt

categorias = ['Producto A', 'Producto B', 'Producto C']
ventas = [150, 230, 190]

plt.figure()
plt.bar(categorias, ventas, color=['skyblue', 'lightcoral', 'lightgreen'])
plt.title('Ventas por Producto')
plt.xlabel('Producto')
plt.ylabel('Unidades Vendidas')
# plt.show()
print("Gráfico de Barras generado (simulado).")
Ventas por Producto (Título)
Eje X: Producto | Eje Y: Unidades Vendidas
(Barras para Producto A (150), B (230), C (190) con colores skyblue, lightcoral, lightgreen)

Histogramas: plt.hist()

Los histogramas son útiles para visualizar la distribución de un conjunto de datos numéricos. Agrupan los datos en "bins" (intervalos) y muestran la frecuencia (cuántos datos caen en cada bin) como barras.

Se crean con plt.hist(datos, bins=N).

  • datos: El array o secuencia de números.
  • bins: El número de intervalos o una secuencia que defina los bordes de los intervalos.
  • Otros parámetros: color, edgecolor (color del borde de las barras), alpha (transparencia).
import matplotlib.pyplot as plt
import numpy as np

# Datos de ejemplo (edades)
np.random.seed(0) # Para reproducibilidad
edades = np.random.randint(18, 65, size=100)

plt.figure()
plt.hist(edades, bins=10, color='salmon', edgecolor='black')
plt.title('Distribución de Edades')
plt.xlabel('Edad')
plt.ylabel('Frecuencia')
# plt.show()
print("Histograma generado (simulado).")
Distribución de Edades (Título)
Eje X: Edad | Eje Y: Frecuencia
(Histograma con 10 bins de color salmón y bordes negros, mostrando la distribución de 100 edades aleatorias entre 18 y 64)

Gráficos de Dispersión: plt.scatter()

Los gráficos de dispersión (scatter plots) son ideales para visualizar la relación entre dos variables numéricas. Cada punto en el gráfico representa una observación.

Se crean con plt.scatter(x, y).

  • Puedes personalizar el color (c), tamaño (s), y transparencia (alpha) de los puntos.
  • El argumento c puede ser un solo color, una secuencia de colores, o incluso una variable numérica que se mapeará a un mapa de colores (cmap).
import matplotlib.pyplot as plt
import numpy as np

np.random.seed(1)
altura = np.random.normal(170, 10, 100) # Media 170, std 10
peso = altura * 0.45 - 10 + np.random.normal(0, 5, 100)
colores_puntos = np.random.rand(100) # Para variar el color

plt.figure()
plt.scatter(altura, peso, c=colores_puntos, alpha=0.7, cmap='viridis')
plt.title('Relación Altura vs. Peso')
plt.xlabel('Altura (cm)')
plt.ylabel('Peso (kg)')
plt.colorbar(label='Valor Aleatorio de Color') # Muestra la barra de color
# plt.show()
print("Gráfico de Dispersión generado (simulado).")
Relación Altura vs. Peso (Título)
Eje X: Altura (cm) | Eje Y: Peso (kg)
(Nube de puntos mostrando una correlación positiva, con colores variables según 'viridis' y una barra de color)

Sectores (pie) y Área (fill_between)

Gráficos de Sectores (plt.pie()):

Muestran la proporción de diferentes categorías como "rebanadas" de un pastel. Son útiles para mostrar partes de un todo.

Sintaxis: plt.pie(tamaños, labels=etiquetas, autopct='%1.1f%%')

  • tamaños: Los valores numéricos que determinan el tamaño de cada sector.
  • labels: Las etiquetas para cada sector.
  • autopct: Un formato de cadena para mostrar el porcentaje en cada sector (ej. '%1.1f%%' muestra un decimal).
  • Otros: colors, explode (para separar un sector), startangle.
import matplotlib.pyplot as plt

etiquetas_pie = ['Manzanas', 'Naranjas', 'Plátanos', 'Uvas']
cantidades_pie = [40, 30, 20, 10]
explode_pie = (0, 0.1, 0, 0) # "explotar" el segundo sector (Naranjas)

plt.figure()
plt.pie(cantidades_pie, explode=explode_pie, labels=etiquetas_pie, autopct='%1.1f%%', shadow=True, startangle=90)
plt.axis('equal') # Asegura que el pastel sea un círculo.
plt.title('Distribución de Frutas')
# plt.show()
print("Gráfico de Sectores generado (simulado).")
Distribución de Frutas (Título)
(Gráfico de pastel con 4 sectores: Manzanas 40%, Naranjas 30% (explotado), Plátanos 20%, Uvas 10%. Porcentajes mostrados)

Gráficos de Área (plt.fill_between()):

Resaltan la magnitud del cambio a lo largo del tiempo o entre categorías rellenando el área entre una línea y el eje x, o entre dos líneas.

import numpy as np
x_area = np.linspace(0, 10, 100)
y_area1 = np.sin(x_area) + 2
y_area2 = np.cos(x_area) + 1

plt.figure()
plt.plot(x_area, y_area1, 'r-', label='Serie 1')
plt.plot(x_area, y_area2, 'b-', label='Serie 2')
plt.fill_between(x_area, y_area1, y_area2, color='cyan', alpha=0.3, where=(y_area1 > y_area2), interpolate=True)
plt.fill_between(x_area, y_area1, y_area2, color='pink', alpha=0.3, where=(y_area1 <= y_area2), interpolate=True)
plt.title('Gráfico de Área entre Curvas')
plt.legend()
# plt.show()
print("\nGráfico de Área generado (simulado).")
Gráfico de Área entre Curvas (Título)
(Dos líneas, Serie 1 (roja) y Serie 2 (azul).
Área entre ellas coloreada: cyan donde Serie 1 > Serie 2, pink donde Serie 1 <= Serie 2)

Ejemplo: Comparando Categorías

Combinemos lo aprendido para crear un gráfico de barras que compare dos series de datos para diferentes categorías.

import matplotlib.pyplot as plt
import numpy as np

labels = ['Grupo 1', 'Grupo 2', 'Grupo 3', 'Grupo 4']
valores_hombres = [20, 35, 30, 35]
valores_mujeres = [25, 32, 34, 20]

x_pos = np.arange(len(labels))  # Posiciones de las etiquetas
ancho_barra = 0.35  # Ancho de las barras

fig, ax = plt.subplots() # Usamos el enfoque OO para más control
rects1 = ax.bar(x_pos - ancho_barra/2, valores_hombres, ancho_barra, label='Hombres', color='dodgerblue')
rects2 = ax.bar(x_pos + ancho_barra/2, valores_mujeres, ancho_barra, label='Mujeres', color='tomato')

# Añadir etiquetas, título y leyenda
ax.set_ylabel('Puntuaciones')
ax.set_title('Puntuaciones por Grupo y Género')
ax.set_xticks(x_pos)
ax.set_xticklabels(labels)
ax.legend()

fig.tight_layout() # Ajusta el padding para que todo quepa bien
# plt.show()
print("Gráfico de Barras Agrupadas generado (simulado).")
Puntuaciones por Grupo y Género (Título)
Eje Y: Puntuaciones | Eje X: Grupo 1, Grupo 2, Grupo 3, Grupo 4
(Para cada grupo, dos barras: una azul para 'Hombres', una roja para 'Mujeres')
Leyenda visible.

Este ejemplo muestra cómo puedes combinar barras para comparaciones directas dentro de cada categoría.

Resumen: Tipos de Gráficos

Matplotlib te ofrece una variedad de gráficos para representar tus datos.

📊 Barras (bar)
Compara cantidades entre categorías.
🧱 Histogramas (hist)
Muestra distribución de datos numéricos.
ScatterIconPlaceholder Dispersión (scatter)
Relación entre dos variables numéricas.
🥧 Sectores (pie)
Proporciones de un todo.
📉 Área (fill_between)
Magnitud de cambio, comparación entre líneas.
🆚 Comparación
Usa el gráfico adecuado para el tipo de datos y el mensaje que quieres transmitir.

Elegir el tipo de gráfico correcto es fundamental para una visualización de datos efectiva.
