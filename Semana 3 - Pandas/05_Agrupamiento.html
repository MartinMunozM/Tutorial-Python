Agrupamiento: .groupby()

Una de las funcionalidades más poderosas de Pandas es la capacidad de agrupar datos con el método .groupby(). Este método sigue el paradigma "Split-Apply-Combine" (Dividir-Aplicar-Combinar):

  1. Dividir (Split): Los datos se dividen en grupos basados en algún criterio (los valores de una o más columnas).
  2. Aplicar (Apply): Se aplica una función a cada grupo de forma independiente (ej. calcular la suma, media, conteo).
  3. Combinar (Combine): Los resultados de la aplicación de la función se combinan en una nueva estructura de datos.

La sintaxis básica es df.groupby('NombreDeColumna'). Esto crea un objeto DataFrameGroupBy, sobre el cual luego aplicas funciones de agregación.

import pandas as pd

data = {'Categoria': ['A', 'B', 'A', 'B', 'A', 'C'],
        'Ventas': [100, 150, 120, 200, 110, 50],
        'Cantidad': [10, 5, 8, 12, 9, 3]}
df = pd.DataFrame(data)
print("DataFrame original:\n", df, "\n")

# Agrupar por 'Categoria'
agrupado_por_categoria = df.groupby('Categoria')

# Ahora podemos aplicar funciones de agregación a este objeto agrupado
suma_ventas_por_categoria = agrupado_por_categoria['Ventas'].sum()
print("Suma de Ventas por Categoría:\n", suma_ventas_por_categoria)

Estadísticas Agregadas Comunes

Una vez que tienes un objeto DataFrameGroupBy, puedes aplicar varias funciones de agregación para calcular estadísticas para cada grupo:

  • .sum(): Suma de los valores.
  • .mean(): Media (promedio).
  • .median(): Mediana.
  • .min(): Valor mínimo.
  • .max(): Valor máximo.
  • .count(): Número de valores no nulos.
  • .size(): Número total de elementos (incluyendo nulos).
  • .std(): Desviación estándar.
  • .var(): Varianza.
import pandas as pd
data = {'Tipo': ['Fruta', 'Verdura', 'Fruta', 'Verdura', 'Fruta'],
        'Precio': [10, 5, 12, 7, 8],
        'Stock': [100, 150, 80, 120, 90]}
df = pd.DataFrame(data)

agrupado_por_tipo = df.groupby('Tipo')

print("Precio medio por Tipo:\n", agrupado_por_tipo['Precio'].mean(), "\n")
print("Stock total por Tipo:\n", agrupado_por_tipo['Stock'].sum(), "\n")
print("Conteo de productos por Tipo:\n", agrupado_por_tipo.size())

Si no especificas una columna después de .groupby(), la función de agregación se aplicará a todas las columnas numéricas compatibles.

Agregación Múltiple: .agg()

El método .agg() (o su alias .aggregate()) es muy potente porque te permite aplicar múltiples funciones de agregación a la vez, e incluso diferentes funciones a diferentes columnas.

Puedes pasar una lista de funciones (como strings o funciones de NumPy) o un diccionario donde las claves son nombres de columna y los valores son las funciones a aplicar.

import pandas as pd
import numpy as np

df_agg = pd.DataFrame({
    'Grupo': ['X', 'Y', 'X', 'Y', 'X', 'Y'],
    'Valor1': [10, 15, 12, 18, 11, 16],
    'Valor2': [100, 50, 80, 60, 90, 70]
})
agrupado = df_agg.groupby('Grupo')

# Aplicar múltiples agregaciones a todas las columnas numéricas
multi_agg = agrupado.agg(['sum', 'mean', np.std])
print("Múltiples agregaciones por grupo:\n", multi_agg, "\n")

# Aplicar diferentes agregaciones a diferentes columnas
agg_especifico = agrupado.agg({
    'Valor1': 'mean',         # Media para Valor1
    'Valor2': ['sum', 'min']  # Suma y mínimo para Valor2
})
print("Agregaciones específicas por columna:\n", agg_especifico)

Agrupamiento Múltiple y Ordenamiento

Agrupamiento por Múltiples Columnas:

Puedes agrupar por más de una columna pasando una lista de nombres de columna a .groupby(). Esto crea un índice jerárquico (MultiIndex) en el resultado.

import pandas as pd
data = {'Region': ['Norte', 'Norte', 'Sur', 'Norte', 'Sur', 'Sur'],
        'Producto': ['A', 'B', 'A', 'A', 'B', 'B'],
        'Ventas': [10, 20, 15, 12, 25, 18]}
df_multi = pd.DataFrame(data)

agrupado_multi = df_multi.groupby(['Region', 'Producto'])
ventas_por_region_producto = agrupado_multi['Ventas'].sum()
print("Ventas por Región y Producto:\n", ventas_por_region_producto, "\n")

Salida esperada:

Ventas por Región y Producto:
Region  Producto
Norte   A           22
        B           20
Sur     A           15
        B           43
Name: Ventas, dtype: int64

Ordenamiento de Resultados Agrupados:

Puedes ordenar los resultados de una agregación usando .sort_values().

# Ordenar el resultado anterior por ventas de forma descendente
ventas_ordenadas = ventas_por_region_producto.sort_values(ascending=False)
print("Ventas ordenadas (descendente):\n", ventas_ordenadas)

Salida esperada:

Ventas ordenadas (descendente):
Region  Producto
Sur     B           43
Norte   A           22
        B           20
Sur     A           15
Name: Ventas, dtype: int64

Si el resultado de .groupby().agg() es un DataFrame, también puedes usar .sort_values(by='NombreDeColumna').
