Selección de Columnas

Una de las operaciones más básicas es seleccionar una o varias columnas de un DataFrame.

Seleccionar una sola columna: Puedes hacerlo usando corchetes [] con el nombre de la columna. Esto devuelve una Serie.

import pandas as pd

data = {'Nombre': ['Ana', 'Luis', 'Sofia'],
        'Edad': [28, 34, 22],
        'Ciudad': ['México', 'Bogotá', 'Lima']}
df = pd.DataFrame(data)
print("DataFrame original:\n", df, "\n")

# Seleccionar la columna 'Nombre'
columna_nombre = df['Nombre']
print("Columna 'Nombre' (es una Serie):\n", columna_nombre)
print(type(columna_nombre))

Salida esperada:

DataFrame original: Nombre Edad Ciudad 0 Ana 28 México 1 Luis 34 Bogotá 2 Sofia 22 Lima Columna 'Nombre' (es una Serie): 0 Ana 1 Luis 2 Sofia Name: Nombre, dtype: object <class 'pandas.core.series.Series'>

Seleccionar múltiples columnas: Pasa una lista de nombres de columnas dentro de los corchetes [[]]. Esto devuelve un nuevo DataFrame.

# Seleccionar las columnas 'Nombre' y 'Ciudad'
multiples_columnas = df[['Nombre', 'Ciudad']]
print("\nColumnas 'Nombre' y 'Ciudad' (es un DataFrame):\n", multiples_columnas)
print(type(multiples_columnas))

Salida esperada:

Columnas 'Nombre' y 'Ciudad' (es un DataFrame): Nombre Ciudad 0 Ana México 1 Luis Bogotá 2 Sofia Lima <class 'pandas.core.frame.DataFrame'>
También puedes usar la notación de punto (df.Nombre) para seleccionar una columna si el nombre de la columna es una cadena válida en Python y no coincide con un método o atributo existente del DataFrame. Sin embargo, se prefiere la notación de corchetes df['Nombre'] por ser más robusta y clara.
🏷️

Selección por Etiquetas: .loc[]

El accesor .loc[] se utiliza para seleccionar datos basándose en las etiquetas de las filas y/o columnas.

Sintaxis común: df.loc[etiqueta_fila, etiqueta_columna]

  • etiqueta_fila: Puede ser una sola etiqueta, una lista de etiquetas, o un slice de etiquetas.
  • etiqueta_columna: Similar, para columnas.

Si tu DataFrame tiene un índice numérico por defecto (0, 1, 2...), .loc[] aún lo trata como etiquetas.

import pandas as pd

data = {'A': [10, 20, 30], 'B': [40, 50, 60], 'C': [70, 80, 90]}
df_loc = pd.DataFrame(data, index=['fila1', 'fila2', 'fila3'])
print("DataFrame con índice de etiqueta:\n", df_loc, "\n")

# Seleccionar la fila 'fila2'
print("Fila 'fila2':\n", df_loc.loc['fila2'])

# Seleccionar el valor en 'fila1', columna 'B'
print("\nValor en ['fila1', 'B']:", df_loc.loc['fila1', 'B'])

# Seleccionar múltiples filas y columnas
print("\nFilas ['fila1', 'fila3'] y Columnas ['A', 'C']:\n", 
      df_loc.loc[['fila1', 'fila3'], ['A', 'C']])

# Slicing con etiquetas (ambos extremos son INCLUIDOS)
print("\nSlicing de filas 'fila1' a 'fila2':\n", df_loc.loc['fila1':'fila2'])

Salida esperada:

DataFrame con índice de etiqueta:
        A   B   C
fila1  10  40  70
fila2  20  50  80
fila3  30  60  90 

Fila 'fila2':
A    20
B    50
C    80
Name: fila2, dtype: int64

Valor en ['fila1', 'B']: 40

Filas ['fila1', 'fila3'] y Columnas ['A', 'C']:
        A   C
fila1  10  70
fila3  30  90

Slicing de filas 'fila1' a 'fila2':
        A   B   C
fila1  10  40  70
fila2  20  50  80
Cuando haces slicing con .loc[] usando etiquetas, el índice final sí se incluye en el resultado, a diferencia del slicing estándar de Python o NumPy donde el final es exclusivo.

Selección por Posición: .iloc[]

El accesor .iloc[] se utiliza para seleccionar datos basándose en su posición entera (índice numérico, como en las listas de Python o arrays de NumPy).

Sintaxis común: df.iloc[posicion_fila, posicion_columna]

  • Siempre usa enteros o slices de enteros.
  • El índice final en un slice no se incluye (comportamiento estándar de Python).
import pandas as pd
data = {'X': ['a', 'b', 'c', 'd'], 'Y': [1, 2, 3, 4], 'Z': [True, False, True, False]}
df_iloc = pd.DataFrame(data, index=['r1', 'r2', 'r3', 'r4'])
print("DataFrame original:\n", df_iloc, "\n")

# Seleccionar la primera fila (posición 0)
print("Primera fila (df_iloc.iloc[0]):\n", df_iloc.iloc[0])

# Seleccionar el valor en la posición (1, 2) -> fila 1, columna 2
print("\nValor en posición (1, 2):", df_iloc.iloc[1, 2])

# Slicing: primeras 2 filas y columnas de la posición 0 a la 1 (excluida la 2)
print("\nSlicing df_iloc.iloc[:2, 0:2]:\n", df_iloc.iloc[:2, 0:2])

Salida esperada:

DataFrame original:
     X  Y      Z
r1  a  1   True
r2  b  2  False
r3  c  3   True
r4  d  4  False 

Primera fila (df_iloc.iloc[0]):
X       a
Y       1
Z    True
Name: r1, dtype: object

Valor en posición (1, 2): False

Slicing df_iloc.iloc[:2, 0:2]:
     X  Y
r1  a  1
r2  b  2
.loc[] es para etiquetas, .iloc[] es para posiciones enteras. ¡No los confundas!
⚖️

Filtrado Condicional y Máscaras Booleanas

Una de las tareas más poderosas es filtrar filas de un DataFrame basándose en condiciones.

Esto se hace creando una máscara booleana (una Serie de True/False) y luego usándola para indexar el DataFrame. Solo se seleccionarán las filas donde la máscara sea True.

import pandas as pd

data = {'Nombre': ['Ana', 'Luis', 'Sofia', 'Carlos'],
        'Edad': [28, 34, 22, 45],
        'Puntos': [150, 200, 120, 250]}
df_filter = pd.DataFrame(data)
print("DataFrame original:\n", df_filter, "\n")

# Filtrar personas mayores de 30 años
mascara_edad = df_filter['Edad'] > 30
print("Máscara de edad (>30):\n", mascara_edad, "\n")
print("Personas mayores de 30:\n", df_filter[mascara_edad], "\n")

# Filtrar personas con más de 180 puntos
# Se puede hacer directamente sin crear la variable máscara
altos_puntos = df_filter[df_filter['Puntos'] > 180]
print("Personas con más de 180 puntos:\n", altos_puntos)

# Combinar condiciones: Edad > 25 Y Puntos < 200
# Usa & para AND, | para OR, y ~ para NOT. ¡No olvides los paréntesis!
condicion_combinada = (df_filter['Edad'] > 25) & (df_filter['Puntos'] < 200)
print("\nPersonas con Edad > 25 Y Puntos < 200:\n", df_filter[condicion_combinada])

Salida esperada:

DataFrame original:
   Nombre  Edad  Puntos
0     Ana    28     150
1    Luis    34     200
2   Sofia    22     120
3  Carlos    45     250 

Máscara de edad (>30):
0    False
1     True
2    False
3     True
Name: Edad, dtype: bool 

Personas mayores de 30:
   Nombre  Edad  Puntos
1    Luis    34     200
3  Carlos    45     250 

Personas con más de 180 puntos:
   Nombre  Edad  Puntos
1    Luis    34     200
3  Carlos    45     250

Personas con Edad > 25 Y Puntos < 200:
  Nombre  Edad  Puntos
0    Ana    28     150
Cuando combines múltiples condiciones booleanas, usa los operadores bitwise & (AND), | (OR), ~ (NOT) y asegúrate de encerrar cada condición individual entre paréntesis debido a la precedencia de operadores.

Resumen: Indexación y Filtrado

¡Dominar la selección de datos es clave en Pandas!

🏛️ Columnas
df['Col'] (Serie)
df[['Col1', 'Col2']] (DataFrame)
🏷️ .loc[] (Etiquetas)
df.loc[fila_etq, col_etq]
Slicing incluye el final.
🔢 .iloc[] (Posición)
df.iloc[fila_pos, col_pos]
Slicing excluye el final.
✅ Máscaras Booleanas
df[df['Col'] > val]
Filtrado condicional.
🔗 Condiciones Múltiples
Usa & (AND), | (OR) con paréntesis. df[(cond1) & (cond2)]
🎯 Precisión
Elige el método correcto (loc vs iloc) según necesites seleccionar por etiqueta o posición.

Con estas técnicas, puedes extraer exactamente los subconjuntos de datos que necesitas para tu análisis.
