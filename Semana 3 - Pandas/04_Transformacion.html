Aplicar Funciones: .apply() y lambda

El método .apply() es muy versátil. Te permite aplicar una función a lo largo de un eje de un DataFrame (filas o columnas) o a cada elemento de una Serie.

A menudo se usa en conjunto con funciones lambda para transformaciones rápidas y concisas.

import pandas as pd
import numpy as np

data = {'Precios': [100, 150, 200, 90],
        'Descuento': [0.1, 0.05, 0.2, 0.15]}
df = pd.DataFrame(data)
print("DataFrame original:\n", df, "\n")

# Aplicar una función lambda a una Serie (columna 'Precios')
precios_con_iva = df['Precios'].apply(lambda x: x * 1.16)
print("Precios con IVA (apply a Serie):\n", precios_con_iva, "\n")

# Aplicar una función definida a todo el DataFrame (por columna, axis=0 por defecto)
def rango_columna(col):
    return col.max() - col.min()

rangos = df.apply(rango_columna) # o df.apply(rango_columna, axis=0)
print("Rango de cada columna (apply a DataFrame):\n", rangos)

# Aplicar a lo largo de las filas (axis=1)
df['Precio_Final'] = df.apply(lambda fila: fila['Precios'] * (1 - fila['Descuento']), axis=1)
print("\nDataFrame con Precio_Final (apply con axis=1):\n", df)

Creando Nuevas Columnas

Crear nuevas columnas en un DataFrame es una tarea muy común. Hay varias formas de hacerlo:

1. Asignando un escalar o una lista/Serie:

import pandas as pd
data = {'Producto': ['A', 'B', 'C'], 'Cantidad': [10, 20, 15]}
df = pd.DataFrame(data)

# Nueva columna con un valor escalar
df['EnStock'] = True 
print("Con columna 'EnStock':\n", df, "\n")

# Nueva columna a partir de una lista (debe tener la misma longitud)
precios_unitarios = [5.0, 2.5, 8.0]
df['PrecioUnitario'] = precios_unitarios
print("Con columna 'PrecioUnitario':\n", df, "\n")

2. Derivada de otras columnas:

# Nueva columna 'Total' calculada a partir de otras
df['Total'] = df['Cantidad'] * df['PrecioUnitario']
print("Con columna 'Total':\n", df)

También puedes usar df.assign(NuevaCol = ...) que devuelve un nuevo DataFrame con la columna añadida, sin modificar el original (útil para encadenar operaciones).

Reemplazar Valores: .replace()

El método .replace() es útil para reemplazar valores en una Serie o en todo un DataFrame.

Puedes reemplazar un valor específico, una lista de valores, o usar un diccionario para mapear valores antiguos a nuevos.

import pandas as pd
import numpy as np

serie = pd.Series([1, 2, 3, 99, 5, -999])
print("Serie original:\n", serie, "\n")

# Reemplazar un solo valor
print("Reemplazar 99 por np.nan:\n", serie.replace(99, np.nan), "\n")

# Reemplazar múltiples valores con un solo valor
print("Reemplazar [99, -999] por 0:\n", serie.replace([99, -999], 0), "\n")

# Reemplazar usando un diccionario (mapeo)
print("Reemplazar {1: 100, 2: 200}:\n", serie.replace({1: 100, 2: 200}))

Cambio de Tipos: .astype()

A veces necesitas cambiar el tipo de dato de una columna (Serie) o de todo un DataFrame. El método .astype() te permite hacer esto.

Puedes convertir a tipos como int, float, str (object), bool, datetime64, etc.

import pandas as pd

df = pd.DataFrame({
    'Edad_str': ['25', '30', '22', '40'],
    'Puntaje_float': [88.5, 92.0, 75.5, 99.0],
    'Activo_int': [1, 0, 1, 1]
})
print("DataFrame original y sus tipos:\n", df)
print(df.dtypes, "\n")

# Cambiar 'Edad_str' a entero
df['Edad_int'] = df['Edad_str'].astype(int)

# Cambiar 'Puntaje_float' a entero (trunca decimales)
df['Puntaje_int'] = df['Puntaje_float'].astype(int)

# Cambiar 'Activo_int' a booleano
df['Activo_bool'] = df['Activo_int'].astype(bool)

print("DataFrame con tipos cambiados:\n", df)
print(df.dtypes)

Ten cuidado al convertir tipos. Por ejemplo, convertir un float a int truncará la parte decimal. Si intentas convertir una cadena que no representa un número a un tipo numérico (ej. 'hola'.astype(int)), obtendrás un ValueError.

Resumen: Transformaciones y Operaciones

Modificar y enriquecer tus datos es una parte fundamental del análisis.

🎨 .apply()
Aplica funciones a Series/DataFrames. df.apply(func, axis=0/1)
🥷 lambda con .apply()
Transformaciones rápidas y en línea.
➕ Nuevas Columnas
df['Nueva'] = ...
Basadas en escalares, listas o otras columnas.
🔁 .replace()
Reemplaza valores específicos o mapeados.
🏷️ .astype()
Cambia el tipo de dato de una columna. df['Col'].astype(nuevo_tipo)

Estas herramientas te permiten limpiar, preparar y enriquecer tus DataFrames para análisis más profundos.
