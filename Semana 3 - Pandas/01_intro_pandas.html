¿Qué es Pandas y Por Qué Usarlo?

Pandas es una librería de Python de código abierto, rápida, potente, flexible y fácil de usar, construida sobre NumPy. Está diseñada específicamente para el análisis y manipulación de datos.

Es la herramienta de facto para trabajar con datos tabulares (como hojas de cálculo o tablas de bases de datos) en Python.

¿Por qué es tan importante?

  • Estructuras de Datos Flexibles: Introduce dos estructuras de datos principales: Series (1D) y DataFrame (2D), que son muy intuitivas para trabajar con datos etiquetados.
  • Manejo de Datos Faltantes: Ofrece herramientas robustas para encontrar, rellenar o eliminar datos faltantes (NaN).
  • Carga y Escritura de Datos: Puede leer y escribir datos fácilmente desde y hacia una gran variedad de formatos (CSV, Excel, SQL, JSON, etc.).
  • Manipulación de Datos: Permite remodelar, filtrar, agrupar, combinar y transformar datos de manera eficiente.
  • Integración: Se integra perfectamente con otras librerías del ecosistema científico de Python como NumPy, Matplotlib y Scikit-learn.

Si vas a realizar cualquier tipo de análisis de datos en Python, Pandas será tu mejor aliado.

Por convención, Pandas se importa con el alias pd: 
import pandas as pd
📊

Estructura 1D: Series

Una Serie de Pandas es como un array de NumPy unidimensional, pero con una característica adicional importante: tiene un índice etiquetado. Esto significa que cada elemento de la Serie tiene una etiqueta (índice) que no tiene por qué ser numérica (aunque por defecto lo es, comenzando en 0).

Creación de Series:

Puedes crear una Serie a partir de una lista, un array de NumPy, o un diccionario.

import pandas as pd
import numpy as np # A menudo se usa con NumPy

# Desde una lista
datos_lista = [10, 20, 30, 40]
serie1 = pd.Series(data=datos_lista)
print("Serie desde lista:\n", serie1)

# Desde una lista con índice personalizado
indices_personalizados = ['a', 'b', 'c', 'd']
serie2 = pd.Series(data=datos_lista, index=indices_personalizados)
print("\nSerie con índice personalizado:\n", serie2)

# Indexación y operaciones básicas
print("\nElemento en índice 'b':", serie2['b'])
print("Multiplicar por 2:\n", serie2 * 2)

Salida esperada:

Serie desde lista: 0 10 1 20 2 30 3 40 dtype: int64 Serie con índice personalizado: a 10 b 20 c 30 d 40 dtype: int64 Elemento en índice 'b': 20 Multiplicar por 2: a 20 b 40 c 60 d 80 dtype: int64

Las Series son la base para las columnas de un DataFrame.

🗂️

Estructura 2D: DataFrame

Un DataFrame es la estructura de datos principal de Pandas. Es una tabla bidimensional, similar a una hoja de cálculo o una tabla SQL, donde los datos están organizados en filas y columnas con etiquetas.

Cada columna de un DataFrame es, en esencia, una Serie de Pandas.

Creación de DataFrames:

Hay muchas formas de crear un DataFrame. Aquí veremos algunas comunes:

1. Desde un diccionario de listas o Series: Las claves del diccionario se convierten en nombres de columna.

import pandas as pd

datos_dict = {
    'Nombre': ['Ana', 'Luis', 'Sofia'],
    'Edad': [28, 34, 22],
    'Ciudad': ['México', 'Bogotá', 'Lima']
}
df1 = pd.DataFrame(datos_dict)
print("DataFrame desde diccionario:\n", df1)

Salida esperada:

DataFrame desde diccionario: Nombre Edad Ciudad 0 Ana 28 México 1 Luis 34 Bogotá 2 Sofia 22 Lima

2. Desde una lista de diccionarios: Cada diccionario representa una fila.

datos_lista_dict = [
    {'Producto': 'Manzana', 'Precio': 50},
    {'Producto': 'Banana', 'Precio': 30},
    {'Producto': 'Naranja', 'Precio': 40, 'Origen': 'MX'} # Puede tener claves diferentes
]
df2 = pd.DataFrame(datos_lista_dict)
print("\nDataFrame desde lista de diccionarios:\n", df2)

Salida esperada (notar el NaN para Origen en las primeras filas):

DataFrame desde lista de diccionarios: Producto Precio Origen 0 Manzana 50 NaN 1 Banana 30 NaN 2 Naranja 40 MX

Pronto veremos cómo leer datos desde archivos CSV, que es una forma muy común de obtener DataFrames.

Inspeccionando un DataFrame

Una vez que tienes un DataFrame, querrás explorarlo para entender su estructura y contenido. Pandas ofrece varios métodos y atributos útiles:

  • .head(n=5): Muestra las primeras n filas (5 por defecto).
  • .tail(n=5): Muestra las últimas n filas (5 por defecto).
  • .shape: Devuelve una tupla con (número_de_filas, número_de_columnas).
  • .info(): Proporciona un resumen conciso del DataFrame, incluyendo el tipo de dato de cada columna, valores no nulos, y uso de memoria.
  • .describe(): Genera estadísticas descriptivas (como conteo, media, desviación estándar, mínimo, máximo, cuartiles) para las columnas numéricas.
  • .columns: Muestra los nombres de las columnas.
  • .index: Muestra la información del índice de las filas.
  • .dtypes: Muestra el tipo de dato de cada columna.
import pandas as pd

datos = {
    'ColumnaA': [1, 2, 3, 4, 5, 6],
    'ColumnaB': ['x', 'y', 'z', 'x', 'y', 'a'],
    'ColumnaC': [10.1, 20.2, 30.3, np.nan, 50.5, 60.6]
}
df = pd.DataFrame(datos)

print("--- .head(3) ---")
print(df.head(3))

print("\n--- .shape ---")
print(df.shape)

print("\n--- .info() ---")
df.info()

print("\n--- .describe() ---")
print(df.describe())
