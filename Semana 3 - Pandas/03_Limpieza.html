Datos Faltantes: El Problema

Los datos del mundo real raramente son perfectos. A menudo, encontrarás conjuntos de datos con valores faltantes o nulos. En Pandas, estos valores suelen representarse como NaN (Not a Number), que proviene de NumPy (np.nan).

¿Por qué importan los datos faltantes?

  • Análisis Incorrectos: Muchos cálculos estadísticos (como la media) pueden dar resultados incorrectos o errores si no se manejan los NaN.
  • Modelos Ineficaces: Los algoritmos de machine learning a menudo no pueden procesar datos con valores faltantes o su rendimiento se ve afectado negativamente.
  • Visualizaciones Engañosas: Los gráficos pueden distorsionarse si no se tratan los NaN.

Por lo tanto, la detección y el manejo adecuado de datos faltantes es un paso crucial en el preprocesamiento de datos.

import pandas as pd
import numpy as np # Para usar np.nan

datos_sucios = {
    'A': [1, 2, np.nan, 4, 5],
    'B': ['x', np.nan, 'z', 'w', np.nan],
    'C': [10, 11, 12, 13, 14]
}
df_sucio = pd.DataFrame(datos_sucios)
print("DataFrame con datos faltantes (NaN):\n", df_sucio)

Salida esperada:

DataFrame con datos faltantes (NaN): A B C 0 1.0 x 10 1 2.0 NaN 11 2 NaN z 12 3 4.0 w 13 4 5.0 NaN 14

Detección de Nulos: .isnull() y .notnull()

Pandas ofrece dos métodos principales para detectar valores nulos:

  • .isnull(): Devuelve un DataFrame (o Serie) booleano del mismo tamaño que el original, donde True indica un valor nulo (NaN) y False indica un valor no nulo.
  • .notnull(): Es lo opuesto a .isnull(). Devuelve True para valores no nulos y False para nulos.
import pandas as pd
import numpy as np
data = {'Col1': [1, np.nan, 3], 'Col2': ['a', 'b', np.nan]}
df = pd.DataFrame(data)
print("DataFrame original:\n", df, "\n")

# Detectar nulos
print("df.isnull():\n", df.isnull(), "\n")

# Contar nulos por columna
print("Conteos de nulos por columna (df.isnull().sum()):\n", df.isnull().sum())

Salida esperada:

0   1.0    a
1   NaN    b
2   3.0  NaN 

df.isnull():
    Col1   Col2
0  False  False
1   True  False
2  False   True 

Conteos de nulos por columna (df.isnull().sum()):

Col2    1
dtype: int64

Puedes usar .isnull().any() para ver si hay algún nulo en una columna/fila, o .isnull().sum().sum() para el total de nulos en el DataFrame.

Manejando Nulos: Eliminación e Imputación

Hay dos estrategias principales para tratar con datos faltantes:

  1. Eliminación: Quitar las filas o columnas que contienen valores nulos.
  2. Imputación: Rellenar los valores nulos con algún valor estimado.

1. Eliminación con .dropna():

  • df.dropna(): Elimina cualquier fila que contenga al menos un valor nulo.
  • df.dropna(axis=1): Elimina cualquier columna que contenga al menos un valor nulo.
  • df.dropna(how='all'): Elimina filas/columnas solo si todos sus valores son nulos.
  • df.dropna(thresh=N): Mantiene filas/columnas que tienen al menos N valores no nulos.
  • inplace=True: Modifica el DataFrame original en lugar de devolver uno nuevo.
import pandas as pd
import numpy as np
df_original = pd.DataFrame({'X': [1, 2, np.nan, 4], 'Y': ['a', np.nan, np.nan, 'd']})
print("Original:\n", df_original, "\n")

print("dropna() (elimina filas con algún NaN):\n", df_original.dropna(), "\n")
print("dropna(axis=1) (elimina columnas con algún NaN):\n", df_original.dropna(axis=1))

2. Imputación con .fillna():

Rellena los valores NaN con un valor específico, la media, la mediana, la moda, o usando métodos de propagación (method='ffill' para rellenar hacia adelante, method='bfill' para rellenar hacia atrás).

df_para_rellenar = pd.DataFrame({'Temp': [25, np.nan, 28, np.nan, 30]})
print("Original para rellenar:\n", df_para_rellenar, "\n")

# Rellenar con un valor específico (ej. 0)
print("fillna(0):\n", df_para_rellenar.fillna(0), "\n")

# Rellenar con la media de la columna
media_temp = df_para_rellenar['Temp'].mean()
print("fillna con la media ({media_temp}):\n", df_para_rellenar.fillna(media_temp))

La elección entre eliminar o imputar depende del contexto: la cantidad de datos faltantes, la naturaleza de los datos y el impacto en el análisis posterior.

Manejo de Duplicados

Otro problema común son las filas duplicadas en un DataFrame. Pandas facilita su detección y eliminación.

  • .duplicated(subset=None, keep='first'): Devuelve una Serie booleana indicando si cada fila es un duplicado de una fila anterior.
    • subset: Lista de nombres de columna a considerar para identificar duplicados. Por defecto, todas las columnas.
    • keep:
      • 'first': Marca duplicados como True excepto por la primera aparición. (Default)
      • 'last': Marca duplicados como True excepto por la última aparición.
      • False: Marca todas las apariciones de duplicados como True.
  • .drop_duplicates(subset=None, keep='first', inplace=False): Devuelve un DataFrame con las filas duplicadas eliminadas. Los argumentos son similares a .duplicated().
import pandas as pd
data_duplicados = {
    'ID': [1, 2, 1, 3, 2, 4],
    'Valor': ['A', 'B', 'A', 'C', 'B', 'D'],
    'Extra': ['x1', 'y1', 'x2', 'z1', 'y2', 'w1']
}
df_dups = pd.DataFrame(data_duplicados)
print("DataFrame con duplicados:\n", df_dups, "\n")

# Detectar duplicados basados en todas las columnas
print("df_dups.duplicated():\n", df_dups.duplicated(), "\n")

# Detectar duplicados basados solo en la columna 'ID'
print("df_dups.duplicated(subset=['ID']):\n", df_dups.duplicated(subset=['ID']), "\n")

# Eliminar duplicados (considerando todas las columnas, manteniendo la primera aparición)
df_sin_dups = df_dups.drop_duplicates()
print("DataFrame sin duplicados (todas las cols):\n", df_sin_dups, "\n")

# Eliminar duplicados basados en 'ID', manteniendo la última aparición
df_sin_dups_id = df_dups.drop_duplicates(subset=['ID'], keep='last')
print("DataFrame sin duplicados por 'ID' (keep='last'):\n", df_sin_dups_id)

Resumen: Limpieza de Datos

La limpieza de datos es un paso esencial antes del análisis.

❓ Datos Faltantes
Comunes en datos reales, impactan análisis. Representados por NaN.
🕵️ .isnull()
Detecta NaN (devuelve booleano). df.isnull().sum() para contar.
🗑️ .dropna()
Elimina filas/columnas con NaN. (axis, how, thresh)
🩹 .fillna()
Rellena NaN con valor, media, mediana, etc. (method='ffill'/'bfill')
👯 .duplicated()
Detecta filas duplicadas. (subset, keep)
✂️ .drop_duplicates()
Elimina filas duplicadas. (subset, keep)

Manejar correctamente los datos faltantes y duplicados asegura la calidad y fiabilidad de tus análisis.
