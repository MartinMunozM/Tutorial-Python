Tutorial: Tomando Decisiones en Python

Aprende a usar if, elif, y else para controlar el flujo de tu programa

🚦

¿Por Qué Tomar Decisiones?

¡Hola! En la vida real, constantemente tomamos decisiones basadas en ciertas condiciones: "Si llueve, llevo paraguas", "Si tengo hambre, como algo".

En programación, también necesitamos que nuestros programas tomen decisiones para comportarse de manera diferente según la situación. Para esto, usamos las estructuras condicionales.

Python nos ofrece las palabras clave if, elif (que significa "else if"), y else para construir estas "ramas de decisión" en nuestro código.

Piensa en los condicionales como bifurcaciones en un camino: según una condición, tu programa tomará una ruta u otra.
❓

La Sentencia if y Condiciones

La estructura if es la más fundamental. Le dice a Python: "Si una condición es verdadera, entonces ejecuta este bloque de código".

La sintaxis es:

if condicion:
    # Código a ejecutar si la condición es True
    # ¡Este código DEBE estar indentado (con espacios o tabulaciones)!

Una condición es una expresión que Python evalúa como True (verdadero) o False (falso). Usamos operadores de comparación para crear estas condiciones:

  • == : Igual a
  • != : No igual a (diferente de)
  • > : Mayor que
  • < : Menor que
  • >= : Mayor o igual que
  • <= : Menor o igual que

Ejemplo:

edad = 18
if edad >= 18:
    print("Eres mayor de edad.")

Salida esperada (si ejecutas este código):

Eres mayor de edad.
¡La indentación es CRUCIAL en Python! El código que pertenece al bloque if debe estar sangrado (generalmente 4 espacios). Si no lo haces, Python dará un error (IndentationError).
↔️

Añadiendo una Alternativa: else

¿Qué pasa si la condición del if es falsa? A veces queremos que se ejecute un código diferente en ese caso. Para esto, usamos else.

La sintaxis es:

if condicion:
    # Código si la condición es True
else:
    # Código si la condición es False

Ejemplo:

temperatura = 15
if temperatura > 25:
    print("Hace calor, ¡ponte ropa ligera!")
else:
    print("No hace tanto calor, quizás necesites una chaqueta.")

Salida esperada:

No hace tanto calor, quizás necesites una chaqueta.

Aquí, como temperatura (15) no es mayor que 25, la condición del if es False, y se ejecuta el código dentro del bloque else.

🚦

Múltiples Condiciones: elif

A veces tenemos más de dos posibilidades. Para esto, usamos elif (contracción de "else if").

Puedes tener tantos elif como necesites. Python los evaluará en orden. Tan pronto como encuentre una condición True, ejecutará su bloque y saltará el resto de elif y el else (si existe).

Sintaxis:

if condicion1:
    # Código si condicion1 es True
elif condicion2:
    # Código si condicion1 es False Y condicion2 es True
elif condicion3:
    # ... y así sucesivamente
else:
    # Código si TODAS las condiciones anteriores son False (opcional)

Ejemplo:

nota = 75
if nota >= 90:
    print("Sobresaliente")
elif nota >= 70:
    print("Notable")
elif nota >= 50:
    print("Aprobado")
else:
    print("Suspenso")

Salida esperada:

Notable

Se imprimirá "Notable" porque nota >= 90 es falso, pero nota >= 70 es verdadero.

🔗

Operadores Lógicos: and, or, not

A veces necesitas combinar varias condiciones. Para esto, Python nos da los operadores lógicos:

  • and (Y lógico): La expresión completa es True solo si ambas condiciones a sus lados son True. 
    edad = 25
tiene_carnet = True
if edad >= 18 and tiene_carnet:
    print("Puedes conducir.")

    Salida:

    Puedes conducir.

  • or (O lógico): La expresión completa es True si al menos una de las condiciones a sus lados es True. 
    es_fin_de_semana = True
es_festivo = False
if es_fin_de_semana or es_festivo:
    print("¡Día libre!")

    Salida:

    ¡Día libre!

  • not (NO lógico): Invierte el valor de verdad de una condición. Si la condición era True, not condicion es False, y viceversa. 
    esta_lloviendo = False
if not esta_lloviendo:
    print("No está lloviendo, ¡vamos a pasear!")

    Salida:

    No está lloviendo, ¡vamos a pasear!

🤝

Combinando Operadores Lógicos

Puedes combinar múltiples operadores lógicos en una sola condición. Python los evaluará siguiendo un orden de precedencia (not primero, luego and, y finalmente or). Sin embargo, es una muy buena práctica usar paréntesis () para agrupar condiciones y hacer que tu intención sea clara.

Ejemplo:

edad = 20
es_estudiante = True
tiene_descuento_especial = False

# Descuento si (es menor de 25) O (es estudiante Y tiene descuento especial)
if (edad < 25) or (es_estudiante and tiene_descuento_especial):
    print("Aplicas para algún tipo de descuento.")
else:
    print("No aplicas para descuentos por ahora.")

Salida esperada:

Aplicas para algún tipo de descuento.

En este caso, edad < 25 es True. Como es una condición or, el resultado total es True.

¡Usa paréntesis siempre que tengas dudas sobre el orden de evaluación o para hacer tu código más legible!
🪆

Condicionales Anidados (if dentro de if)

Puedes poner una estructura if/elif/else completa dentro de otro bloque if, elif, o else. Esto se llama anidamiento.

Es útil cuando una segunda decisión depende del resultado de la primera.

Ejemplo:

usuario_registrado = True
es_admin = False

if usuario_registrado:
    print("Bienvenido, usuario.")
    if es_admin:  # Este 'if' está anidado
        print("Tienes acceso al panel de administración.")
    else:
        print("No tienes permisos de administrador.")
else:
    print("Por favor, regístrate para continuar.")

Salida esperada:

Bienvenido, usuario. No tienes permisos de administrador.
¡Cuidado con la indentación! Cada nivel de anidamiento requiere una indentación adicional. Demasiados niveles pueden hacer el código difícil de leer.
🚧

La Palabra Clave pass

A veces, necesitas una estructura condicional pero aún no sabes qué código irá dentro de ese bloque. Si dejas un bloque vacío, Python dará un error.

Para evitar esto, puedes usar la palabra clave pass. pass es una operación nula: no hace nada, simplemente actúa como un marcador de posición.

Ejemplo:

temperatura = 30

if temperatura > 35:
    print("Alerta: ¡Calor extremo!")
elif temperatura > 25:
    # Aún no he decidido qué hacer aquí, pero no quiero un error
    pass 
else:
    print("Temperatura normal.")
print("Fin del reporte de temperatura.")

Si la temperatura es 30, se evaluará el elif, se ejecutará pass (no hace nada), y el programa continuará, imprimiendo "Fin del reporte de temperatura.".

👀

Comparando Booleanos: Buenas Prácticas

Cuando tienes una variable booleana (que ya es True o False), no necesitas compararla explícitamente con True o False en tus condiciones if.

Forma menos recomendada (pero funciona):

esta_listo = True
if esta_listo == True:
    print("¡El sistema está listo!")

Forma recomendada (más "Pythónica" y legible):

esta_listo = True
if esta_listo:  # Directamente, si es True, se ejecuta
    print("¡El sistema está listo!")

hay_errores = False
if not hay_errores: # Si NO hay_errores (es decir, si hay_errores es False)
    print("No hay errores.")

Esto hace tu código más conciso y fácil de entender.

Resumen de Condicionales

¡Excelente trabajo! Has aprendido a controlar el flujo de tu programa con condicionales:

❓ if condicion:
Ejecuta si True. ¡Indentación!
↔️ else:
Alternativa si if es False.
🚦 elif condicion:
Múltiples condiciones en secuencia.
⚙️ Comparadores
==, !=, >, <, >=, <=
🔗 Operadores Lógicos
and, or, not. Usa ().
🪆 Anidados
if dentro de if. Cuida la indentación.
🚧 pass
Marcador para bloques vacíos.
👀 Booleanos
if var: o if not var:

Los condicionales son una de las herramientas más poderosas en programación. ¡Practícalos mucho!
