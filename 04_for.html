Tutorial: Repitiendo Tareas con Bucles for

Aprende a iterar sobre secuencias y automatizar procesos en Python

🔁

¿Qué es un Bucle for?

¡Hola de nuevo! A veces, necesitamos repetir una acción varias veces. Imagina que quieres saludar a 10 personas diferentes o procesar cada elemento de una lista de compras.

Escribir el mismo código una y otra vez sería tedioso y propenso a errores. Para esto, usamos bucles. El bucle for en Python es una herramienta poderosa para iterar (recorrer) sobre secuencias de elementos.

El bucle for de Python es más parecido a una estructura "for-each". En lugar de manejar contadores manualmente, Python toma directamente cada elemento de una secuencia.

La sintaxis básica es:

for variable_temporal in secuencia:
    # Código a ejecutar para cada elemento
    # ¡Este código DEBE estar indentado!

Donde variable_temporal tomará el valor de cada elemento de la secuencia en cada repetición (iteración) del bucle.

📋

Iterando sobre Listas

Una de las secuencias más comunes sobre las que iteramos son las listas.

Ejemplo: Saludar a amigos en una lista.

amigos = ["Ana", "Luis", "Sofía"]

for nombre in amigos:
    print(f"¡Hola, {nombre}!")

Salida esperada:

¡Hola, Ana! ¡Hola, Luis! ¡Hola, Sofía!

Ejemplo: Sumar números en una lista.

numeros = [10, 20, 30, 5]
suma_total = 0

for numero in numeros:
    suma_total += numero  # Forma corta de: suma_total = suma_total + numero

print(f"La suma total es: {suma_total}")

Salida esperada:

La suma total es: 65
🔢

La Función range()

¿Qué pasa si quieres repetir una acción un número específico de veces, sin tener una lista predefinida? La función range() es perfecta para esto, ya que genera una secuencia de números.

  • range(stop): Genera números desde 0 hasta stop-1. 
    for i in range(5):  # i tomará valores 0, 1, 2, 3, 4
    print(f"Iteración número: {i}")

    Salida:

    Iteración número: 0
    Iteración número: 1
    Iteración número: 2
    Iteración número: 3
    Iteración número: 4

  • range(start, stop): Genera números desde start hasta stop-1. 
    for num in range(2, 6): # num tomará valores 2, 3, 4, 5
    print(num)

    Salida:

    2
    3
    4
    5

  • range(start, stop, step): Genera números desde start hasta stop-1, incrementando en step. 
    for x in range(1, 10, 2): # x tomará valores 1, 3, 5, 7, 9
    print(x)

    Salida:

    1
    3
    5
    7
    9

Recuerda que range(stop) y range(start, stop) llegan hasta stop-1.
🔡

Iterando sobre Cadenas de Texto

Las cadenas de texto también son secuencias (de caracteres), ¡así que puedes iterar sobre ellas con un bucle for!

palabra = "Python"
for letra in palabra:
    print(letra)

Salida esperada:

P y t h o n
Esto es útil para procesar cada carácter de un texto, por ejemplo, para contar vocales o convertir a mayúsculas.
🛑

break: Saliendo Temprano del Bucle

A veces, mientras iteras, encuentras lo que buscas o se cumple una condición que hace innecesario continuar. La palabra clave break te permite salir del bucle for inmediatamente.

Ejemplo: Buscar un número y detenerse al encontrarlo.

numeros = [1, 5, 8, 12, 7, 3]
numero_buscado = 12

for num in numeros:
    print(f"Probando con: {num}")
    if num == numero_buscado:
        print("¡Número encontrado!")
        break  # Salimos del bucle
print("Búsqueda terminada.")

Salida esperada:

Probando con: 1 Probando con: 5 Probando con: 8 Probando con: 12 ¡Número encontrado! Búsqueda terminada.

El bucle se detendrá después de encontrar el 12.

⏭️

continue: Saltando a la Siguiente Iteración

La palabra clave continue salta el resto del código dentro de la iteración actual y pasa directamente a la siguiente iteración del bucle.

Ejemplo: Imprimir solo números impares.

numeros = [1, 2, 3, 4, 5, 6, 7]

for num in numeros:
    if num % 2 == 0:  # Si el número es par
        continue     # Saltamos el print para este número par
    print(f"Número impar: {num}")

Salida esperada:

Número impar: 1 Número impar: 3 Número impar: 5 Número impar: 7
✔️

La Cláusula else en Bucles for

Python tiene una característica un poco inusual: los bucles for pueden tener una cláusula else.

El bloque else se ejecuta solo si el bucle termina normalmente, es decir, si recorre todos los elementos sin encontrar una sentencia break.

Ejemplo: Usando else para saber si un break ocurrió.

items = ["manzana", "banana", "naranja"]
item_a_buscar = "pera" # Prueba también cambiando a "banana"

for item in items:
    if item == item_a_buscar:
        print(f"¡Encontrado: {item_a_buscar}!")
        break
else:
    # Este bloque se ejecuta solo si el bucle for terminó sin un break
    print(f"{item_a_buscar} no encontrado en la lista.")

Si item_a_buscar es "pera", la salida será:

pera no encontrado en la lista.

Si item_a_buscar es "banana", la salida será:

¡Encontrado: banana!
(y el else no se ejecuta).

🪆

Bucles for Anidados

Al igual que con los condicionales if, puedes poner un bucle for dentro de otro. Esto se llama bucles anidados.

El bucle interno se ejecutará completamente por cada iteración del bucle externo.

Ejemplo: Imprimir combinaciones de colores y tallas.

colores = ["Rojo", "Azul"]
tallas = ["S", "M", "L"]

for color in colores:      # Bucle externo
    for talla in tallas:  # Bucle interno
        print(f"Producto: {color}, Talla: {talla}")

Salida esperada:

Producto: Rojo, Talla: S Producto: Rojo, Talla: M Producto: Rojo, Talla: L Producto: Azul, Talla: S Producto: Azul, Talla: M Producto: Azul, Talla: L
Los bucles anidados pueden volverse lentos si las secuencias son muy grandes. ¡Úsalos con cuidado!
