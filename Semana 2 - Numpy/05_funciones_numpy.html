Lógica Condicional: np.where()

La función np.where(condicion, valor_si_true, valor_si_false) es extremadamente útil. Es como una versión vectorizada de una sentencia if-else que opera sobre arrays.

Devuelve un nuevo array donde los elementos se eligen de valor_si_true o valor_si_false según el valor de condicion (que debe ser un array booleano).

import numpy as np

arr = np.array([1, 5, 10, 15, 20])
print("Array original:", arr)

# Reemplazar valores mayores a 10 con -1, y los demás con 0
resultado_where = np.where(arr > 10, -1, 0)
print("Resultado con np.where:", resultado_where)

# También puedes pasar arrays como valor_si_true y valor_si_false
arr_true = np.array([100, 200, 300, 400, 500])
arr_false = np.array([11, 22, 33, 44, 55])
resultado_where_arrays = np.where(arr <= 10, arr_true, arr_false)
print("np.where con arrays:", resultado_where_arrays)

Salida esperada:

Array original: [ 1 5 10 15 20] Resultado con np.where: [ 0 0 0 -1 -1] np.where con arrays: [100 200 300 44 55]
np.where es muy eficiente para la manipulación condicional de datos en arrays.

Elementos Únicos: np.unique()

La función np.unique(arr) devuelve los elementos únicos y ordenados de un array.

Es útil para identificar los distintos valores presentes en tus datos o para eliminar duplicados.

import numpy as np

datos_con_duplicados = np.array([1, 2, 2, 3, 4, 4, 4, 5, 1])
print("Datos originales:", datos_con_duplicados)

unicos = np.unique(datos_con_duplicados)
print("Elementos únicos:", unicos)

# También puede devolver los índices de la primera aparición y las cuentas
unicos, indices, cuentas = np.unique(datos_con_duplicados, return_index=True, return_counts=True)
print("\nUnicos con detalles:", unicos)
print("Índices de primera aparición:", indices)
print("Conteos de cada único:", cuentas)

Salida esperada:

Datos originales: [1 2 2 3 4 4 4 5 1] Elementos únicos: [1 2 3 4 5] Unicos con detalles: [1 2 3 4 5] Índices de primera aparición: [0 1 3 4 7] Conteos de cada único: [2 2 1 3 1]

Valores Especiales: isnan(), isinf()

En cálculos numéricos, a veces te encontrarás con valores especiales como NaN (Not a Number) o inf (infinito).

  • np.nan: Representa un valor indefinido o no representable (ej. resultado de 0/0).
  • np.inf: Representa infinito positivo (ej. resultado de 1/0).
  • -np.inf: Representa infinito negativo.

NumPy proporciona funciones para detectar estos valores:

  • np.isnan(arr): Devuelve un array booleano indicando si cada elemento es NaN.
  • np.isinf(arr): Devuelve un array booleano indicando si cada elemento es infinito (positivo o negativo).
  • np.isfinite(arr): Devuelve un array booleano indicando si cada elemento es finito (ni NaN ni inf).
import numpy as np

datos_con_especiales = np.array([1.0, np.nan, 3.0, np.inf, -np.inf, 5.0])
print("Datos originales:", datos_con_especiales)

print("\n¿Es NaN?:", np.isnan(datos_con_especiales))
print("¿Es Infinito?:", np.isinf(datos_con_especiales))
print("¿Es Finito?:", np.isfinite(datos_con_especiales))

# Filtrar valores NaN
datos_sin_nan = datos_con_especiales[~np.isnan(datos_con_especiales)] # ~ es el operador NOT bitwise
print("\nDatos sin NaN:", datos_sin_nan)

Salida esperada:

Datos originales: [ 1. nan 3. inf -inf 5.] ¿Es NaN?: [False True False False False False] ¿Es Infinito?: [False False False True True False] ¿Es Finito?: [ True False True False False True] Datos sin NaN: [ 1. 3. inf -inf 5.]
Las comparaciones directas con np.nan (ej. x == np.nan) siempre devuelven False. ¡Usa siempre np.isnan() para verificar si hay NaN!

Ordenando Arrays: sort() y argsort()

NumPy ofrece funciones eficientes para ordenar arrays:

  • np.sort(arr): Devuelve una copia ordenada del array. No modifica el original.
  • arr.sort(): Ordena el array en el lugar (modifica el original) y no devuelve nada (devuelve None).
  • np.argsort(arr): Devuelve los índices que ordenarían el array original.
import numpy as np

desordenado = np.array([3, 1, 4, 1, 5, 9, 2, 6])
print("Array desordenado:", desordenado)

# np.sort() devuelve una copia ordenada
ordenado_copia = np.sort(desordenado)
print("Ordenado (copia):", ordenado_copia)
print("Original después de np.sort():", desordenado) # No cambia

# arr.sort() ordena en el lugar
desordenado_para_inplace = np.array([5, 0, 8])
desordenado_para_inplace.sort()
print("\nOrdenado en el lugar:", desordenado_para_inplace)

# np.argsort() devuelve índices
indices_orden = np.argsort(desordenado) # Usa el 'desordenado' original
print("\nÍndices para ordenar 'desordenado':", indices_orden)
print("Array ordenado usando argsort:", desordenado[indices_orden])

Salida esperada:

Array desordenado: [3 1 4 1 5 9 2 6] Ordenado (copia): [1 1 2 3 4 5 6 9] Original después de np.sort(): [3 1 4 1 5 9 2 6] Ordenado en el lugar: [0 5 8] Índices para ordenar 'desordenado': [1 3 6 0 2 4 7 5] Array ordenado usando argsort: [1 1 2 3 4 5 6 9]

Para arrays 2D, puedes especificar axis=0 (ordenar por columnas) o axis=1 (ordenar por filas).

Generación de Números Aleatorios

NumPy tiene un potente módulo np.random para generar números aleatorios, lo cual es esencial para simulaciones, muestreo, y muchas otras tareas.

  • np.random.rand(d0, d1, ..., dn): Crea un array de la forma dada con números aleatorios de una distribución uniforme entre 0 (inclusivo) y 1 (exclusivo).
  • np.random.randn(d0, d1, ..., dn): Similar, pero de una distribución normal estándar (media 0, desviación estándar 1).
  • np.random.randint(bajo, alto=None, size=None, dtype=int): Devuelve enteros aleatorios desde bajo (inclusivo) hasta alto (exclusivo). Si alto es None, el rango es [0, bajo). size define la forma del array resultante.
  • np.random.seed(semilla): Fija la "semilla" del generador de números aleatorios. Esto es crucial para la reproducibilidad: si usas la misma semilla, obtendrás la misma secuencia de números "aleatorios" cada vez que ejecutes el código.
import numpy as np

# Para resultados reproducibles
np.random.seed(42) 

# Array 1D de 5 números aleatorios uniformes
aleatorios_uniformes = np.random.rand(5)
print("Aleatorios uniformes [0,1):", aleatorios_uniformes)

# Matriz 2x3 de números aleatorios normales
aleatorios_normales = np.random.randn(2, 3)
print("\nAleatorios normales (media 0, std 1):\n", aleatorios_normales)

# 3 enteros aleatorios entre 1 (incluido) y 7 (excluido)
enteros_aleatorios = np.random.randint(1, 7, size=3)
print("\nEnteros aleatorios [1,7):", enteros_aleatorios)

Salida esperada (los números exactos dependerán de la semilla, pero deberían ser los mismos si ejecutas esto con la semilla 42):

Aleatorios uniformes [0,1): [0.37454012 0.95071431 0.73199394 0.59865848 0.15601864] Aleatorios normales (media 0, std 1): [[ 0.05808361 0.86617615 0.60111501] [-0.70807258 -0.02058848 0.26551159]] Enteros aleatorios [1,7): [3 5 1]
