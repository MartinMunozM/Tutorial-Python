Tutorial: print() en Python

Aprende a usar la función más importante de Python paso a paso

¡Bienvenida y tu Primera Línea de Código!

¡Hola! Estás a punto de aprender una de las herramientas más fundamentales en programación: la función print().

Imagina que quieres que tu computadora "diga" algo en la pantalla. En Python, un lenguaje de programación muy popular y amigable, usamos print() para esto.

¿Qué es print()? Es una instrucción que le dice a Python: "muestra esto en la pantalla".

No te preocupes si "función" o "instrucción" suenan complicados. Por ahora, piensa en print() como un comando mágico que hace aparecer texto.

En este tutorial aprenderás:

  • ✅ Cómo usar print() básico para texto y números.
  • ✅ Imprimir múltiples cosas y variables.
  • ✅ Personalizar la salida con sep y end.
  • ✅ Usar f-strings para formato moderno.
  • ✅ Evitar errores comunes.
💬

Imprimir Texto (Cadenas)

La forma más común de usar print() es para mostrar un mensaje de texto. En programación, al texto lo llamamos "cadena de caracteres" o "string".

Para decirle a Python que algo es texto, debemos encerrarlo entre comillas dobles (" ") o comillas simples / apóstrofes (' ').

Ejemplo con comillas dobles:

print("¡Hola, Mundo!")

Salida esperada:

¡Hola, Mundo!

Ejemplo con comillas simples:

print('Python es divertido')

Salida esperada:

Python es divertido
¡Cuidado! Si empiezas con comillas dobles, debes terminar con comillas dobles. Lo mismo con las simples. Mezclarlas (ej. print("texto')) causará un error.
🔢

Imprimir Números

Con print() también puedes mostrar números. ¡No necesitas comillas para los números!

Ejemplo con un número entero:

print(42)

Salida esperada:

42

Ejemplo con un número decimal (flotante):

print(3.14159)

Salida esperada:

3.14159
Diferencia importante:
• print("42") imprime el texto "42".
• print(42) imprime el número 42.
¡El resultado se ve igual en la pantalla, pero internamente Python los trata de forma diferente! Esto es crucial para operaciones matemáticas.
➕📦

Múltiples Elementos y Variables

Puedes imprimir varias cosas a la vez separándolas con comas (,) dentro de print(). Python automáticamente pondrá un espacio entre ellas.

print("Mi nombre es", "Juan", "y tengo", 25, "años")

Salida esperada:

Mi nombre es Juan y tengo 25 años

Las variables son como cajitas donde guardas información. Puedes imprimir su contenido:

nombre = "María"
edad = 30
ciudad = "Guadalajara"

print("Hola, soy", nombre)
print("Vivo en", ciudad, "y tengo", edad, "años.")

Salida esperada:

Hola, soy María Vivo en Guadalajara y tengo 30 años.
Al imprimir variables, no uses comillas alrededor de sus nombres. print(nombre) está bien, pero print("nombre") imprimiría la palabra "nombre".
✨

Caracteres Especiales y Comillas

Puedes agregar saltos de línea, tabulaciones y otros efectos usando "caracteres de escape" (que comienzan con una barra invertida \):

print("Primera línea\nSegunda línea")
print("Texto con\ttabulación")

Salida esperada:

Primera línea Segunda línea Texto con tabulación

¿Cómo imprimir comillas dentro de un texto?

  1. Alternar comillas: Si tu texto usa comillas dobles, defínelo con simples (o viceversa).
  2. Escapar la comilla: Usa \" para una comilla doble o \' para una simple.
# Opción 1: Alternar
print('Ella dijo: "¡Hola Python!"')

# Opción 2: Escapar
print("Él respondió: \"Python es genial\".")

Salida esperada:

Ella dijo: "¡Hola Python!" Él respondió: "Python es genial".

Principales caracteres de escape:

  • \n : Salto de línea.
  • \t : Tabulación horizontal.
  • \\ : Barra invertida.
  • \' : Comilla simple.
  • \" : Comilla doble.
⚙️

Personalizando print(): Parámetro sep

Cuando imprimes múltiples elementos, Python usa un espacio como separador por defecto. ¡Puedes cambiar esto con el parámetro sep (de "separator")!

print("manzanas", "naranjas", "plátanos") # Separador por defecto: espacio
print("manzanas", "naranjas", "plátanos", sep=", ")
print("Día", "Mes", "Año", sep="-")
print(1, 2, 3, sep="") # Sin separador

Salida esperada:

manzanas naranjas plátanos manzanas, naranjas, plátanos Día-Mes-Año 123
El parámetro sep debe ir después de todos los elementos que quieres imprimir, y el valor que le asignes (lo que va después del =) debe ser texto (entre comillas).
🔚

Personalizando print(): Parámetro end

Por defecto, print() siempre termina con un salto de línea. Puedes cambiar esto con el parámetro end.

print("Hola", end=" ") # Termina con un espacio
print("Mundo")      # Este print continúa en la misma línea y luego salta

print("Contando: ", end="") # No agrega nada al final
print(1, end=", ")
print(2, end=", ")
print(3, end="!"\n") # Termina con "!" y un salto de línea explícito
print("Siguiente línea.")

Salida esperada:

Hola Mundo Contando: 1, 2, 3! Siguiente línea.
¡Experimenta! Prueba con diferentes valores para end, como end="***" o end=" --> ". Si quieres que la siguiente impresión comience en una nueva línea después de usar end con algo que no sea un salto de línea, puedes añadir \n a tu end, como en end="!!!\n".
🎯

F-strings: La Forma Moderna y Fácil

Las f-strings (cadenas formateadas) son una forma súper elegante y moderna de combinar texto con el valor de variables o expresiones. ¡Solo pon una f justo antes de las comillas de apertura!

nombre = "Carlos"
edad = 22
promedio = 8.5

print(f"Hola {nombre}, tienes {edad} años.")
print(f"Tu promedio es {promedio}.")
print(f"El año que viene tendrás {edad + 1} años.")

Salida esperada:

Hola Carlos, tienes 22 años. Tu promedio es 8.5. El año que viene tendrás 23 años.
¿Por qué son geniales las f-strings?
• Son más fáciles de leer que otras formas de formateo.
• Puedes poner expresiones y operaciones matemáticas directamente dentro de las llaves { }.
• Es la forma preferida y más moderna en Python para estas tareas.
🚫

Errores Comunes y Cómo Evitarlos

Todos cometemos errores al empezar. Aquí están los más comunes con print():

Error 1: Olvidar los paréntesis
Incorrecto: print "Hola" (Esto era válido en Python 2, ¡pero no en Python 3!)
Correcto: print("Hola")
Error 2: Mezclar tipos de comillas
Incorrecto: print("Hola')
Correcto: print("Hola") o print('Hola')
Error 3: Olvidar comillas para el texto
Incorrecto: print(Hola mundo) (Python pensará que Hola y mundo son variables)
Correcto: print("Hola mundo")
Error 4: Imprimir una variable no definida
Si escribes print(mi_dato) pero mi_dato no existe, Python dará un error NameError.
Consejo profesional: Lee los mensajes de error. Python usualmente te da pistas sobre qué salió mal. Si algo no funciona, revisa siempre las comillas, paréntesis y la ortografía de tus variables.
