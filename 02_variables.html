¿Qué Son las Variables?

¡Bienvenido/a de nuevo! Ahora que sabes cómo mostrar mensajes con print(), vamos a aprender sobre las variables.

Imagina una variable como una caja con una etiqueta. Dentro de la caja, puedes guardar información (datos), y la etiqueta te ayuda a saber qué hay dentro y a encontrarla fácilmente.

En programación, las variables nos permiten:

  • Almacenar datos para usarlos más tarde.
  • Dar nombres significativos a esos datos.
  • Modificar los datos si es necesario.
Piensa en ellas como los "sustantivos" de tu programa: guardan información sobre personas, lugares, números, estados, etc.
✏️

Creando (Declarando) Variables

Crear una variable en Python es muy sencillo. Usamos un nombre para la variable, el signo igual (=), y luego el valor que queremos guardar.

La sintaxis es: nombre_de_la_variable = valor

El signo = aquí no significa "igualdad" como en matemáticas. Significa "asignar". Es como decir: "guarda este valor en la caja con esta etiqueta".

Ejemplos:

edad = 25                  # Guardamos el número 25 en la variable 'edad'
nombre_usuario = "Ana López"     # Guardamos el texto "Ana López" en 'nombre_usuario'
precio_producto = 19.99           # Guardamos el número decimal 19.99 en 'precio_producto'
es_estudiante = True          # Guardamos el valor booleano True en 'es_estudiante'

Una vez que asignas un valor a una variable, Python "recuerda" esa asociación.

📜

Reglas para Nombrar Variables

No puedes usar cualquier nombre para tus variables. Python tiene algunas reglas:

  • Deben empezar con una letra (a-z, A-Z) o un guion bajo (_).
  • Después del primer carácter, pueden contener letras, números (0-9) y guiones bajos.
  • No pueden contener espacios ni caracteres especiales como !, @, #, $, %, etc. (excepto el guion bajo).
  • Python distingue entre mayúsculas y minúsculas. miVariable es diferente de mivariable.
  • No pueden ser iguales a las "palabras reservadas" de Python (palabras que Python usa para sus propias instrucciones, como print, if, for, while, etc.).

Buenas Prácticas (Convenciones):

  • Usa nombres descriptivos: total_compra es mejor que tc.
  • Para nombres de variables con múltiples palabras, la convención más común en Python es usar snake_case: palabras en minúscula separadas por guiones bajos (ej. nombre_del_cliente, tasa_interes_anual).
Nombres Incorrectos: 
# 1erNombre (empieza con número)
# nombre-usuario (contiene guion medio en lugar de guion bajo)
# mi variable (contiene espacio)
# class (es una palabra reservada)
Nombres Correctos: 
primer_nombre
nombre_usuario
mi_variable
_variable_privada # (El guion bajo inicial tiene un significado especial, pero es válido)
🧩

Tipos de Datos Comunes

Las variables pueden guardar diferentes tipos de información. Los más comunes al empezar son:

  • Texto (String - str): Secuencias de caracteres, como palabras o frases. Siempre van entre comillas (" ") o apóstrofes (' '). 
    saludo = "¡Hola, Python!"
ciudad = 'Madrid'
  • Números Enteros (Integer - int): Números completos, sin decimales. 
    cantidad_alumnos = 30
temperatura_actual = -5 # También pueden ser negativos
  • Números Decimales (Float - float): Números con punto decimal. 
    pi_aproximado = 3.14159
peso_kg = 65.7
  • Booleanos (Boolean - bool): Representan valores de verdad: solo pueden ser True o False (¡con la primera letra en mayúscula!). Son muy útiles para tomar decisiones en el código. 
    esta_lloviendo = True
es_fin_de_semana = False
Python es un lenguaje de tipado dinámico. Esto significa que no necesitas declarar explícitamente el tipo de una variable. Python lo deduce automáticamente del valor que le asignas.
📤

Usando Variables con print()

¡Claro! Una vez que tienes datos guardados en variables, ¡quieres usarlos! La forma más sencilla de ver el contenido de una variable es con print().

nombre_curso = "Programación con Python"
numero_lecciones = 10

print(nombre_curso)
print(numero_lecciones)

Salida esperada:

Programación con Python 10

También puedes combinar texto y variables en un print(), por ejemplo, usando f-strings (¡recuerda el módulo anterior!):

articulo = "libro"
precio = 25.50

print(f"El precio del {articulo} es de {precio} pesos.")

Salida esperada:

El precio del libro es de 25.5 pesos.
🔄

Reasignar Valores a las Variables

El valor guardado en una variable no es fijo para siempre. Puedes cambiarlo (reasignarlo) cuantas veces necesites.

puntos = 100
print(f"Puntos iniciales: {puntos}")

# El jugador gana más puntos
puntos = 150
print(f"Puntos después de ganar: {puntos}")

# También puedes modificar una variable basándote en su valor actual
puntos = puntos + 20  # Incrementa los puntos en 20
print(f"Puntos finales: {puntos}")

Salida esperada:

Puntos iniciales: 100 Puntos después de ganar: 150 Puntos finales: 170
Python ejecuta el código línea por línea. Cuando reasignas una variable, el valor anterior se reemplaza por el nuevo.
🔩

Parseo: Convirtiendo Tipos de Datos

A veces, necesitas convertir un dato de un tipo a otro. Esto se llama parseo o "type casting".

Por ejemplo, cuando obtienes datos de un usuario (lo veremos en el siguiente paso), ¡siempre vienen como texto (string)!, incluso si el usuario escribe un número.

Python nos da funciones para convertir:

  • int(valor): Intenta convertir valor a un número entero.
  • float(valor): Intenta convertir valor a un número decimal.
  • str(valor): Convierte valor a una cadena de texto.

Ejemplos:

numero_en_texto = "123"
numero_real = int(numero_en_texto)
print(numero_real + 10) # Ahora podemos hacer operaciones matemáticas

decimal_en_texto = "99.50"
precio_real = float(decimal_en_texto)
print(precio_real * 2)

edad_numero = 25
mensaje = "Tengo " + str(edad_numero) + " años." # Necesitamos str() para unir texto con números usando +
print(mensaje)

Salida esperada:

133 199.0 Tengo 25 años.
Error Común: Intentar convertir un texto que no representa un número válido a int() o float() causará un error (ValueError). Por ejemplo, int("hola") dará error.
⌨️

Recibiendo Datos con input()

Una forma común de hacer tus programas interactivos es pedirle datos al usuario. En Python, usamos la función input().

input() muestra un mensaje al usuario (opcional) y espera a que escriba algo y presione Enter. Importante: ¡input() siempre devuelve lo que el usuario escribe como una cadena de texto (string)!

nombre_usuario = input("Por favor, introduce tu nombre: ")
print(f"¡Hola, {nombre_usuario}! Bienvenido/a.")

edad_texto = input("¿Cuántos años tienes? ")
# edad_texto es un string, incluso si el usuario escribe un número.
# Para usarlo como número, debemos convertirlo (parsearlo):
edad_numero = int(edad_texto)

print(f"El año que viene tendrás {edad_numero + 1} años.")
Si ejecutas este código, el programa se detendrá y esperará tu entrada en la consola. ¡Pruébalo en un entorno Python si tienes uno configurado! (Este tutorial web no puede ejecutar código Python directamente).

Adelanto: Guardar Múltiples Valores (Listas)

Hasta ahora, cada variable guarda un solo valor (un nombre, una edad, un precio).

Pero, ¿qué pasa si quieres guardar varios valores relacionados bajo un mismo nombre? Por ejemplo, una lista de nombres de amigos, o una lista de tareas pendientes.

Para esto, Python tiene algo llamado listas. Una lista se crea usando corchetes [ ] y separando los elementos con comas.

# Una lista de nombres
amigos = ["Ana", "Luis", "Sofía", "David"]

# Una lista de números
calificaciones = [10, 8, 9, 7, 10]

# Una lista puede contener diferentes tipos de datos
mixta = ["Tarea 1", 5, True]

print(amigos)
print(calificaciones)

Salida esperada:

['Ana', 'Luis', 'Sofía', 'David'] [10, 8, 9, 7, 10]
Las listas son increíblemente útiles y poderosas. Aprenderás mucho más sobre ellas (cómo acceder a elementos individuales, modificarlas, etc.) en un módulo dedicado. ¡Esto es solo un pequeño adelanto!
