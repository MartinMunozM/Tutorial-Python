Tutorial: Bucles while en Python

Aprende a repetir acciones mientras una condición sea verdadera

🔄

¿Qué es un Bucle while?

¡Continuamos explorando cómo repetir tareas! Ya conoces el bucle for, que es ideal para iterar sobre secuencias.

El bucle while (que significa "mientras" en inglés) es diferente: repite un bloque de código mientras una condición especificada sea verdadera.

Es útil cuando no sabes de antemano cuántas veces necesitas repetir la acción, sino que la repetición depende de que algo siga siendo cierto.

Piensa en el bucle while como: "Mientras esta luz esté verde, sigue avanzando".

La sintaxis básica es:

while condicion:
    # Código a ejecutar mientras la condición sea True
    # ¡Este código DEBE estar indentado!
    # Es CRUCIAL que algo dentro del bucle eventualmente haga que la condición sea False.
🔢

Un Bucle while Simple: Contador

Un ejemplo clásico es usar un bucle while para contar.

Ejemplo: Contar hasta 5.

contador = 1  # 1. Inicializamos una variable de control

while contador <= 5:  # 2. Condición: mientras contador sea menor o igual a 5
    print(f"El contador es: {contador}")
    contador += 1   # 3. Actualización: incrementamos el contador. ¡MUY IMPORTANTE!

print("¡Bucle terminado!")

Salida esperada:

El contador es: 1 El contador es: 2 El contador es: 3 El contador es: 4 El contador es: 5 ¡Bucle terminado!

Componentes clave:

  1. Inicialización: Se prepara una variable (contador) antes del bucle.
  2. Condición: Se evalúa en cada iteración. Si es True, el bucle continúa.
  3. Actualización: Dentro del bucle, se modifica la variable de control para que, eventualmente, la condición se vuelva False y el bucle termine.
☠️

Peligro: Bucles Infinitos

Si la condición de un bucle while nunca se vuelve falsa, el bucle se ejecutará para siempre. Esto se llama un bucle infinito y puede hacer que tu programa se "cuelgue" o consuma todos los recursos.

Ejemplo de un bucle infinito (¡NO LO EJECUTES SIN SABER CÓMO DETENERLO!):

numero = 10
while numero > 0:
    print("El número sigue siendo positivo...")
    # ¡ERROR! Olvidamos decrementar 'numero'. 
    # 'numero' siempre será 10, y 10 > 0 siempre será True.
Para evitar bucles infinitos: Asegúrate siempre de que la variable o estado que controla la condición del while se modifique dentro del bucle de tal manera que la condición eventualmente se vuelva False.
Si alguna vez ejecutas un bucle infinito en una consola o terminal, usualmente puedes detenerlo presionando Ctrl + C (o Cmd + C en Mac).
🛑

break en Bucles while

Al igual que con los bucles for, puedes usar la palabra clave break para salir de un bucle while prematuramente, incluso si su condición principal sigue siendo verdadera.

Ejemplo: Adivinar un número.

numero_secreto = 7
intentos = 0

while True: # Un bucle "infinito" controlado por un break
    suposicion = int(input("Adivina el número (1-10): "))
    intentos += 1
    if suposicion == numero_secreto:
        print(f"¡Correcto! Lo adivinaste en {intentos} intentos.")
        break # Salimos del bucle porque el usuario adivinó
    elif suposicion < numero_secreto:
        print("Demasiado bajo. Intenta de nuevo.")
    else:
        print("Demasiado alto. Intenta de nuevo.")

Este patrón de while True: con un break interno es común para bucles que deben terminar por una condición específica dentro del cuerpo del bucle.

⏭️

continue en Bucles while

La palabra clave continue también funciona en bucles while. Salta el resto del código de la iteración actual y vuelve a verificar la condición del while para decidir si continuar con la siguiente iteración.

Ejemplo: Procesar solo números positivos ingresados por el usuario.

numeros_positivos_sumados = 0
cantidad_positivos = 0

print("Introduce números. Escribe 0 para terminar.")
while True:
    entrada = int(input("Número: "))
    
    if entrada == 0:
        break # Terminar el bucle si se ingresa 0
        
    if entrada < 0:
        print("Número negativo ignorado.")
        continue # Volver al inicio del bucle, sin sumar ni contar
        
    # Si llegamos aquí, el número es positivo
    numeros_positivos_sumados += entrada
    cantidad_positivos += 1

print(f"Suma de números positivos: {numeros_positivos_sumados}")
print(f"Cantidad de números positivos: {cantidad_positivos}")
¡Cuidado con continue! Asegúrate de que la variable que controla la condición principal del while (o una condición de break) todavía pueda alcanzarse y modificarse adecuadamente, incluso después de un continue, para evitar bucles infinitos.
✔️

La Cláusula else en Bucles while

Al igual que los bucles for, los bucles while también pueden tener una cláusula else.

El bloque else se ejecuta solo si la condición del bucle while se vuelve falsa (y el bucle termina normalmente), no si se sale del bucle con un break.

Ejemplo: Simular un temporizador.

cuenta_regresiva = 3
while cuenta_regresiva > 0:
    print(f"{cuenta_regresiva}...")
    cuenta_regresiva -= 1
    # Si aquí hubiera un 'break' condicional, el 'else' no se ejecutaría si ese break ocurre.
else:
    print("¡Tiempo terminado!") # Se ejecuta porque el while terminó normalmente

Salida esperada:

3... 2... 1... ¡Tiempo terminado!
Esta cláusula else es menos común en bucles while que en bucles for, pero puede ser útil en situaciones específicas para ejecutar código de "limpieza" o finalización solo si el bucle completó su ciclo natural.

Resumen de Bucles while

¡Has aprendido mucho sobre los bucles while!

🔄 Básico
while condicion:
Repite mientras True.
⚙️ Control
Inicializa, condiciona, ¡y actualiza!
☠️ Bucle Infinito
Evítalo asegurando que la condición se vuelva False.
🛑 break
Sale del bucle inmediatamente.
⏭️ continue
Salta a la siguiente iteración.
✔️ else en bucle
Si el bucle termina sin break.
⚠️ Indentación
¡Obligatoria y crucial!

Los bucles while son excelentes cuando el número de iteraciones no se conoce de antemano. ¡Son una herramienta fundamental!
